從岡山站，搭津山線，約一個半小時，來到津山市
目標是每年在櫻花時節有桜まつり並且入選賞櫻百名所之一的津山城

津山城種植了五千多棵櫻樹，
滿開時的一整片粉紅海，
在西日本是屬一屬二名所

每年在櫻花盛開的時節，會有櫻的祭典，夜晚有夜櫻觀賞
古城，屋台，燈光，櫻花，交織成十分夢幻的場景
是此行四個賞夜櫻的晚上，最美的一個場所　（當然跟天氣也有關啦）

搖搖晃晃，經過無數田原風光，昏昏欲睡，

這也是島田莊司作品：犬坊里美的冒險這本書的舞台之一

穿過小小的地下道，通往津山站前
可以看到站前已經立了大大的津山さくらまつり的看板

渡過吉井川，太陽西下，兩岸燈火亮起
而在橋上就可以見到遠遠被櫻花簇擁著的津山城


過了橋後來到了鶴山通
<a href="http://ppt.cc/baqA" rel="nofollow" target="_blank">http://ppt.cc/baqA</a>

津山城算是一個發展很成熟的城，城下町曾經非常大而繁華
不過現在時間約六點多，店家卻大部份已經關閉
商店街也顯得蕭條


津山地區的人孔蓋的主題是河童
在整個鶴山通的兩旁，也處處是河童的縱跡
<a href="http://ppt.cc/jZ,G" rel="nofollow" target="_blank">http://ppt.cc/jZ,G</a>

為什麼津山這個感覺在深山中的城市會和河童扯上關係？
有人認為是因為津山處於內陸，和大阪，江戶的交流主要是靠吉井川的水路交通
而吉井川的水道較為彎曲，行舟不易
因此河童的出現有可能指的就是深闇水性的人
而島田莊司在書中也寫到，這類靠水路而繁盛的都市，常常會有河童傳說
現在每年八月，津山地區也都會舉行河童祭

在登城口石段的右方是津山觀光center，有賣許多相關紀念品和土產
左方是個教堂

因為有桜まつり，路旁有許多的攤販　<a href="http://ppt.cc/Ky,K" rel="nofollow" target="_blank">http://ppt.cc/Ky,K</a>
天色見暗，我們拾級而上　<a href="http://ppt.cc/AXF9" rel="nofollow" target="_blank">http://ppt.cc/AXF9</a>
裡面已經有許多民眾在盡情地享受櫻花盛開的祭典
<a href="http://ppt.cc/pXYQ" rel="nofollow" target="_blank">http://ppt.cc/pXYQ</a>

路旁的屋台，生意非常的好


鶴山公園是原來的津山城郭對外開放後，種植了五千多棵櫻樹，
而成為日本西部屬一屬二賞櫻名所
<a href="http://ppt.cc/3@I3" rel="nofollow" target="_blank">http://ppt.cc/<span class="__cf_email__" data-cfemail="c0f38089f3">[email protected]</span></a>

在受付處，有一代城主森忠敬的像
因為有夜櫻的關係，入城時間延長到夜間十點

付了300 yen的入園費後，正式進入津山城跡
在進入三之丸後有津山城登城紀念板可供合照
攜家帶眷一同出遊的家庭很多

向二之丸這段階梯爬完後，左方就是表門跡
此時天色漸暗，遇上了美麗的魔幻時刻
粉紅的櫻花，打上燈光，配上蔚藍的天，夢幻
<a href="http://ppt.cc/WjlM" rel="nofollow" target="_blank">http://ppt.cc/WjlM</a>　　<a href="http://ppt.cc/z-Nl" rel="nofollow" target="_blank">http://ppt.cc/z-Nl</a>

津山城是一個梯郭式平山城，從三之丸到本丸要一路爬階梯而上
最後終於看到被櫻花環繞的備中櫓了！
這是２００５年才剛復元的櫓，也是津山城唯一的建物
我非常喜歡夜櫻，那閃爍的金光，瀰漫的粉紅，巍峨的古牆，像是仙境一般
<a href="http://ppt.cc/GIHu" rel="nofollow" target="_blank">http://ppt.cc/GIHu</a>

走近發現可以遠眺整個市區，古時的城下町
<a href="http://ppt.cc/OiTX" rel="nofollow" target="_blank">http://ppt.cc/OiTX</a>

只好沿著本丸及二之丸再找找
