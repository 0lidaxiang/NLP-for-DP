一般大家廣為使用到富士山週遭的交通方式是利用富士箱根
周遊票三日券，然而若是不想到箱根而只想到富士五湖，或
者排不出三天假期週遊箱根與富士山
(雖然仍有二日券5000円)，那麼使用這張票便浪費了。

這提供一個較為便宜到富士五湖的方法，就是直接利用富士
急高速バス，從新宿出發往河口湖，現場買票來回各1700円
可停靠的位置包含富士急樂園、河口湖、本栖湖 、 山中湖
等地(價格各有不同，到河口湖是1700円)。

路線圖
<a href="http://tinyurl.com/7st4nbc" rel="nofollow" target="_blank">http://tinyurl.com/7st4nbc</a>

這裡要介紹的除了上述部分，我要分享的內容是在利用富士
急高速バス網站預約訂位、使用WEB回數券折價、還有搭乘
湯瑪士小火車造型巴士。


<span class="f5">WEB回數券折價</span>

WEB回數券一次四張(共5990円)，如果你是結伴而行而且有
信用卡的話，那麼兩個人來回各兩張剛好可以把它用完。
如此一來，往來新宿與河口湖的價格單趟可降為1500円，
來回一共可省下400円。

購買之前要先到富士急高速バス網站申請會員。
<a href="http://www.highwaybus.com" rel="nofollow" target="_blank">http://www.highwaybus.com</a>

選擇新規會員登錄
<a href="http://tinyurl.com/6ufoatx" rel="nofollow" target="_blank">http://tinyurl.com/6ufoatx</a>

然後同意它的一些事項
<a href="http://tinyurl.com/76avnkh" rel="nofollow" target="_blank">http://tinyurl.com/76avnkh</a>
<a href="http://tinyurl.com/86k4x75" rel="nofollow" target="_blank">http://tinyurl.com/86k4x75</a>

進入資料填寫的部分
<a href="http://tinyurl.com/6lvfxpg" rel="nofollow" target="_blank">http://tinyurl.com/6lvfxpg</a>

從上往下分別是
想要的ID (英數4字以上12字以內)
密碼
重覆密碼

名字: 姓 X 名 XX
名子的讀音:
性別
生日
電話
郵遞區號
職業
e-mail
e-mail再次確認
e-mail的種類: 選パソコン
可否收到廣告: 可以選いいえ(不要)


而讀音的部分可以利用這個網站 <a href="http://nihongo.j-talk.com/kanji/" rel="nofollow" target="_blank">http://nihongo.j-talk.com/kanji/</a>
右方的Readings in brackets ，把你的名子輸入後選右下方
的一個轉換符號(在Copy字眼隔壁) ，就可獲得讀音 。
姓名的部分因為有些漢字日文裡面沒有，若堅持使用這系統
的話可使用你姓名的相似字。
郵遞區號與電話可填你住宿的飯店。

之後應該會收到註冊信。
取得帳號後就可選擇首頁上方的WEB回數券購入。
<a href="http://tinyurl.com/7z243zu" rel="nofollow" target="_blank">http://tinyurl.com/7z243zu</a>

這時候依起點與要去的地方分別填入。
<a href="http://tinyurl.com/85dxyz5" rel="nofollow" target="_blank">http://tinyurl.com/85dxyz5</a>

選擇檢索後可獲得以下結果
<a href="http://tinyurl.com/6lo5axg" rel="nofollow" target="_blank">http://tinyurl.com/6lo5axg</a>

選擇購入後可來到以下網頁，下方分別是
信用卡號碼、信用卡上的名字、有效日期
<a href="http://tinyurl.com/84mzr7s" rel="nofollow" target="_blank">http://tinyurl.com/84mzr7s</a>
接下來確認後，就可購入WEB回數券。購入完畢一樣會收到信。


<span class="f5">利用網站預約訂位</span>

首先在首頁選擇要搭乘的系統、路線、起點、終點、日期。
<a href="http://tinyurl.com/7yzdhga" rel="nofollow" target="_blank">http://tinyurl.com/7yzdhga</a>

確定後可看到該日期各班車的狀態，只要不是X符號的表示還有
位置，還可以選擇女性專用座位。
<a href="http://tinyurl.com/7xl45sh" rel="nofollow" target="_blank">http://tinyurl.com/7xl45sh</a>

接下來填入人數，座位的需求(之後還可以改變)，左下方
可同時預約回程班車。
<a href="http://tinyurl.com/7reovxb" rel="nofollow" target="_blank">http://tinyurl.com/7reovxb</a>

最後就是情報資料告訴你這要多少錢。
<a href="http://tinyurl.com/8ylyks3" rel="nofollow" target="_blank">http://tinyurl.com/8ylyks3</a>

預約確認後，可利用右下方的WEB回數券付費。
<a href="http://tinyurl.com/8834zsd" rel="nofollow" target="_blank">http://tinyurl.com/8834zsd</a>

最後會收到完成的信，之後可利用首頁的予約済の方裡面的訂位資料，
或信裡面的連結所連到的頁面，把它列印出來後到現場搭車就可以了。


<span class="f5">湯瑪士小火車造型巴士</span>

富士急高速バス到富士五湖一天分別有六班車是湯瑪士小火車
造型巴士，來回各三班。
官方網站: <a href="http://bus.fujikyu.co.jp/thomaslandex/index.html" rel="nofollow" target="_blank">http://bus.fujikyu.co.jp/thomaslandex/index.html</a>

分別是從新宿出發的1131、1107、1117班車。這裡的到達站是
富士急樂園，但不用擔心，這班車是會繼續開到河口湖之後的
。而回程則是1102、1110、1136號班車，這裡的時刻一樣是列
富士急樂園，可查詢在河口湖等站的開車時刻。只要只定搭乘
這些班車，就可以搭到湯瑪士小火車造型巴士。
<a href="http://tinyurl.com/78wlr8b" rel="nofollow" target="_blank">http://tinyurl.com/78wlr8b</a>
