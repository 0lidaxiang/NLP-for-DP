四月九日 天氣晴時多雲午後陣雨

原訂行程: 銀閣寺-哲學之道-南禪寺-蹴上鐵道-疏水紀念館-圓山公園夜櫻/高台寺夜櫻

實際行程: 銀閣寺-哲學之道-南禪寺-蹴上鐵道-疏水紀念館-花見小路-祇園白川南通夜櫻

          -圓山公園夜櫻-寧寧之道-高台寺夜櫻

一大早小小偷懶晚了些起床，八點吧，老樣子吃了長的一樣的早餐和食，就拉著行李

CHECK OUT來去旁邊的京都塔飯店了。

九時許，幸運地跳上洛100公車，前往第一個目的地─銀閣寺。

不半個小時，我看見滿滿的櫻花懸在枝頭。不愧是板友回報哲學之道北段接銀閣寺道滿開

中的關雪櫻。流水清澈，水中的櫻映出粉藍的天空。不知不覺，自己都有氣質了起來，也

哲學了起來，在這如畫的美景之下!


<a href="http://www.flickr.com/photos/juor2/7079057075/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079057075/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079058863/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079058863/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079061473/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079061473/in/set-72157629819806611/</a>

殺了不少快門後，左轉來到文化遺產─銀閣寺。其實他叫東山慈照寺，本來聽說要學金閣

寺貼金箔，不過好像作罷了XD!。難怪我覺得這銀在哪裡啊。走著走我又發現大敵人─沙

子路(崩潰)，於是只能帶著很複雜的心情走完這裡TAT。


<a href="http://www.flickr.com/photos/juor2/6932993776/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6932993776/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079068641/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079068641/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079072741/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079072741/in/set-72157629819806611/</a>

走啊走的，一直到大豊神社前，其實，櫻花都還是滿開的狀態。後半段雖然比較少些但也

還是有七分咲。走啊走的，突然有人大叫我XD，原來是昨天把我拎去清水寺的新朋友們~

大家果然功課作的都很勤勞，向著繁花茂盛之處前行啊:P。


<a href="http://www.flickr.com/photos/juor2/7079073293/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079073293/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/6933001048/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933001048/in/set-72157629819806611/</a>


然後帶著愉悅的心情，疼痛的右膝蓋，來到了南禪寺，這時我第一個想到的就是午餐啊

!!!心裡想的是順正，眼睛看的是奧丹。於是在一陣迷路之下好不容易，好不容易來到了

排隊人潮不算少的─南禪寺順正。來到京都一定要吃的就是豆腐料理了。在一陣餓肚子的

等待之後，我點了價位比較高的月套餐，因為這趟旅行都在走路，吃的部分大多都是在賞

夜櫻的地方跟lawson就地解決了，所以難得好好吃一頓，就不管價錢了XD。


<a href="http://www.flickr.com/photos/juor2/7079075647/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079075647/in/set-72157629819806611/</a>

不得不說，這豆腐比想像中多樣化，甜點，豆奶，火鍋，烤品等等。雖然我也很想開一鍋

豆漿鍋來捲豆皮啊XD。其實這樣的一組套餐，說真的有點飽~，對一個大男生而言，不過

這是為了接下來的暴走行程作能量儲備啊。


<a href="http://www.flickr.com/photos/juor2/7079076813/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079076813/in/set-72157629819806611/</a>

下午二時許，雲開始多起來，太陽也開始跟著害羞，偶爾透出光芒，也吹起了有點冷的微

春東風。來到的是南禪寺三門，碰巧遇上了特別的參拜，所以就殘念了。不過小小坐在三

門旁發個呆，看看三門旁滿開中的櫻花也很好。還有望著好像是台灣來的中年攝影團，每

台都是全幅等級加上快門線配上全幅鏡架著腳架(什麼無敵珊，滴飼跟溜飼午低的一大票~

好貴!)。


<a href="http://www.flickr.com/photos/juor2/7079081643/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079081643/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079084807/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079084807/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/6933011968/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933011968/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079087661/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079087661/in/set-72157629819806611/</a>

然後來到蹴上鐵道與琵琶湖疏水公園，這裡是因為當初沒設計好，其後發現落差太大的一

段運輸道。其後在櫻花的點綴之下，成為每年春季人潮洶湧的景點之一。不得不配服日本

人在保留許多歷史文物這方面，許多在時代洪流上可能無足輕重的一瞬，在小小的巧思以

及保養之下，成為了後代子孫能夠繼續傳承下去的價值。(突然想起台灣文林苑~有點扯遠

了@@)

不知不覺就走到了盡頭，好吧，老樣子找棵滿開的樹下坐著放空，也許這就是我喜歡一個

人旅行的其中一點吧，想走想停任憑自己。就在這個時候─突然發現我牛仔褲怎麼在很尷

尬，但有點很隱密的地方狂開的裂開= ="。不會是幫路人拍照蹲下來取景太多次造成的吧

(好像才第二天就十幾次了)，真的是屋漏偏逢連夜雨啊，腳跛了褲子還炸開Orz|||。

<a href="http://www.flickr.com/photos/juor2/6933012798/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933012798/in/set-72157629819806611/</a>

結果說到雨，還真的下起傾盆大雨XD!還颳起風來!於是乎找個地方休息一下，點個紅豆年

糕湯配上暖呼呼的綠茶。雖然不是什麼名店，但補充一下血糖打個盹也不錯:P


<a href="http://www.flickr.com/photos/juor2/6933014316/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933014316/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/6933016012/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933016012/in/set-72157629819806611/</a>

雨勢趨緩，清新的空氣和著慢慢散去的雲朵，不經意在岡崎疏水道看見微微透出的耶穌光

。和著清流與掛在枝頭的粉紅花簇，讓人不禁佇留目光。


走向神宮道，遇上騎著單車有點迷失方向的背包客姐姐們，指引方向小聊以後，路過知恩

院向著圓山祗園一帶前行。


<a href="http://www.flickr.com/photos/juor2/7079094349/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079094349/in/set-72157629819806611/</a>

圓山公園，望見了傳說中的枝垂櫻女王，大約有八成咲，但整體氣勢非凡啊!。路上只見佔

位中的日本媽媽跟公司職員們，還有準備上工的攤販店家。我想，就先前往白川南通，晚

些時間再回來這附近解決晚餐。(其實是本來想要吃的店找不到在哪XD!)

<a href="http://www.flickr.com/photos/juor2/7079097233/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079097233/in/set-72157629819806611/</a>

途中走過雨中的八坂神社，走進小路，經過花見小路，可惜沒有巧遇傳說中趕場的美麗藝

妓。於是乎來到了天啊，真的是滿開的白川!!!!!

<a href="http://www.flickr.com/photos/juor2/6933023774/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933023774/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079099983/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079099983/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/6933027608/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933027608/in/set-72157629819806611/</a>



隨著夕陽西下，我被謀殺的快門數也跟著瘋狂上升。美麗的色溫掛著粉白的花朵，掛在河

川旁的町家餐廳。哪天，閒錢夠多一定要這樣浪漫地找個女伴一同用餐，雖然八字在這二

十幾年來都沒有一撇過啊XD!。


<a href="http://www.flickr.com/photos/juor2/6933031178/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933031178/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079120067/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079120067/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/6933033840/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933033840/in/set-72157629819806611/</a>

夜色升起，打著光的盛開櫻花頓時間我居然會看成衛生紙團(囧)，這好像是非常惹人厭的

困擾啊XD!!(此話一出就被吐嘈了XD)


<a href="http://www.flickr.com/photos/juor2/6933038106/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/6933038106/in/set-72157629819806611/</a>
<a href="http://www.flickr.com/photos/juor2/7079114791/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079114791/in/set-72157629819806611/</a>

回到圓山公園找不著在tabelog上預先找好的晚餐地點，那就隨興配著啤酒，在花樹下吃

著屋台裡的許多燒物，何嚐不是一種有趣的體驗?。望著閃亮的櫻女王，今天就敬妳一杯

吧!cheers!看著周遭卸下工作面具的日本上班族們，也是蠻有趣的:p


<a href="http://www.flickr.com/photos/juor2/7079116745/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079116745/in/set-72157629819806611/</a>

在一陣掙扎之後，我還是拎著相機拖著炸掉中的腳，踩著寧寧之道走向高台寺了XD。高台

寺的櫻花說真的不是很多，但最有名的那株垂櫻正滿開著，路上的宣傳看板如此寫著。可

惜了又是一個不能用腳架的地方，所以又挑戰手持的功力啊XD。


<a href="http://www.flickr.com/photos/juor2/7079119213/in/set-72157629819806611/" rel="nofollow" target="_blank">http://www.flickr.com/photos/juor2/7079119213/in/set-72157629819806611/</a>

除了這株垂櫻外，我想就屬這片打光的竹林最引我注目吧:P，有種森嚴但透著希望微光的

氛圍。


最後我又走向清水道旁的206號公車站牌回到飯店，如同前一日，在Lawson血拼那伴著我

夜晚整理照片，還有熱水澡後的夜宵。

(睡前決定隔天一早一定要去樓下泡個湯再出門)
