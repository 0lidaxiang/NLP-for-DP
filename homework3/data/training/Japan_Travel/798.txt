然百名城才進行了不到三分之一，姬路城已經來四次了。
但是賞姬路城的夜櫻這可是頭一次呢。

這也是我們第一次見到被包住的大天守 <a href="http://ppt.cc/~VbQ" rel="nofollow" target="_blank">http://ppt.cc/~VbQ</a>
雖然一開始有點失望
但夜櫻的美　讓我們忘了失望！！

純文字版：

之前幾次來姬路城多半是用Kansai Thru Pass搭山陽電鐵的特急來的，
到了姬路之後就是出車站一路往北走就到了姬路城。
這次除了改搭JR之外，還在東駅前町住了兩晚，
因此發現了姬路市另一個面貌，還有罪惡的みゆき通り商店街。
(大大小小的店該有的都有　不錯好買）
<a href="http://ppt.cc/Baa;" rel="nofollow" target="_blank">http://ppt.cc/Baa;</a>

不過這條商店街也是很標準的晚上七點之後就沒人了。
一樣往北走，稍微左轉一下就會回到大路上，直達姬路城。

護城河上遠遠的就看到開得正漂亮的櫻花。
<a href="http://ppt.cc/Snn_" rel="nofollow" target="_blank">http://ppt.cc/Snn_</a>

自從姬路城包起來整修之後我就沒來過了。
不過今晚的夜櫻是在西之丸庭園裡，包起來的天守應該不是主角。
<a href="http://ppt.cc/~VbQ" rel="nofollow" target="_blank">http://ppt.cc/~VbQ</a>

大門口擺著可愛的姬路城吉祥物
姬路城真是太美了，就算沒有天守，還是有一堆櫓可以當主角。

到了夜間的開放入口處，人不多，
不知道是不是因為天守包起來所以大家興趣缺缺？
我們要進去的時候工作人員說音樂會正好開始囉～請快進去吧～
好期待呢～

來了來了！看見裡面打燈了！  <a href="http://ppt.cc/tk31" rel="nofollow" target="_blank">http://ppt.cc/tk31</a>

夜櫻只開放西之丸庭園，所以往天守的路被封了起來。
我們回轉往西之完前進，
一瞬間就被打在牆上的漂亮櫻花吸引了！好漂亮啊！！
<a href="http://ppt.cc/GLUB" rel="nofollow" target="_blank">http://ppt.cc/GLUB</a>

西之丸庭園的入口擺著夜櫻會的看板。
上面寫著投扇體驗會因雨天中止，真是可惜。

庭園中搭起了一個小小的棚子，  <a href="http://ppt.cc/Vrud" rel="nofollow" target="_blank">http://ppt.cc/Vrud</a>
裡面是Keyboard, violin, 以及日本笛的Trio。
同是吹笛人，雖然覺得日本笛的音色很糟，音準很差，

但是不可否認的就是這反而成了最鮮明的特色。
放開心胸，在夜櫻的美景之中欣賞著面對面的演出，
又是一個一輩子忘不了的旅行記憶。

有去過姬路城的人應該都對西之丸庭園裡的這個小池子有印象吧。
今天晚上燈光把這個池子打得好美好美。

千姬的化妝櫓也是沒有開放的，有點可惜。　<a href="http://ppt.cc/h-I," rel="nofollow" target="_blank">http://ppt.cc/h-I,</a>
不過今天的主角就是這些燦爛的櫻花

這邊的夜櫻是用一直在變色的光源在打的，
所以有一種與其他地方夜櫻不一樣的氣氛。
<a href="http://ppt.cc/rRO3" rel="nofollow" target="_blank">http://ppt.cc/rRO3</a>

這棵樹讓我想起楊牧《星圖》裡描寫的那棵巴特列德梨，
一樹光彩燦爛.......　　<a href="http://ppt.cc/24h4" rel="nofollow" target="_blank">http://ppt.cc/24h4</a>

連路燈都好漂亮。

在出口處又看到了這吉祥物，
原來她叫しろまるひめ（白丸姬？！）
還挺可愛的啦....
在大天守整修前似乎沒有這個吉祥物？

最後再回頭看一下西之丸的方向，
沒蓋到姬路城的章，明早再來！！

如果對於姬路城在整修不知道要不要去
至少至少　夜櫻的部份　還是非常有看頭的啊！
