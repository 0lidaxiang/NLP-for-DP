Day1 (12/29): 台北 → 名古屋

● 在一航廈各區間移動, 有種一直穿越時空的感覺。
● 證照查驗、海關、車票, 過關斬將, 每次來到名古屋, 總是有種氣場很合、手氣很順
的感覺。
● check-in 時領到 b-mobile U300, 依指示換卡、設定、開通, 5min 展開線上生活。

Day2 (12/30): 名古屋 → 高山 (租車: 高山 → 五箇山 → 白川鄉 → 高山)

● 在特急ひだ上寫完二、三十張賀年明信片, 手寫有種特殊的連結感, 很能回味。
● 名古屋発の特急列車 おりた時から高山駅は 雪の中 … (哇, 音節都對得上 XD)
● 再遊五箇山, 終於見到糖霜過多的薑餅屋; 白川鄉四回目, 比起城山展望台, かん町
遊人稀疏, 意境甚佳。


Day3 (12/31): 高山 → 奧飛驒溫泉鄉

● 雪晴, 美得刺骨, 櫻山八幡宮集御朱印, 高山老街找酒。
● 再訪槍見館。泡在貸切溫泉中, 眼前是流水、是白雪、是大吟釀, 啊～～
● 奧飛驒山區沒有除夜鐘, 只有蒲田川的潺潺水聲, 祝大家新年快樂。


Day4 (1/1): 高山 → 金澤

● 在槍見館現煮、現搗、現吃完麻糬, 互道恭喜。
● 元旦假期搭新穗高纜車的人都與西穗高岳緣慳一面, 我們來三次也只見過它一面。
● 到達加賀百萬石城下町, 氣溫10℃, 體感還不如寒流來襲的台北城, 這其中一定有什
麼誤會 @<a class="__cf_email__" data-cfemail="673827" href="/cdn-cgi/l/email-protection">[email protected]</a>


Day5 (1/2): 金澤 → 雨晴 → 高岡 → 金澤

● 站在雨晴站月台上, 沒有其他遊人、沒有氣嵐、沒有立山連峰, 只有滿天烏雲與蝕骨
海風, 在心裡大喊一聲「靠!」
● 瑞龍寺初詣, 高岡大佛再詣。
● 高岡唯一一間營業的郵局, ご当地フォルムカード賣完了, 富山縣版確定摃龜 Orz
● 天氣不好、運氣不好, 只好大吃一頓。日本海的寒鰤真的不是開玩笑的啊 ))))


Day6 (1/3): 金澤 → 白馬

● 金澤市區完全沒有積雪, 放棄三訪兼六園。
● 糸魚川站等車兼午餐 1.5hr 都在看箱根駅伝轉播。
● 轉入大糸線, 進入雪國模式, 南小谷的候車室裡還提供こたつ(暖桌)呢。
● 來到滑雪村, 氣氛像是又出了一次國。


Day7 (1/4): 白馬 (HAKUBA47)

● 本日主題是：翻滾吧, 阿宅!
● 滑雪場裡, 有人像是生來腳上就長了一對 skis (或雪板), 我們則像一直踩到絆馬索
; 不過有教練指導再實際滑(摔)幾趟, 身體逐漸就能記住訣竅, 反正臉都包起來, 就能完
全放下羞恥心了 XD
● wii 忘了告訴我們滑雪需要好肌力。


Day8 (1/5): 白馬 → 長野

● 續戰滑雪場。風雪未減, 視線不良, 一直看不清遠山景色是最大的遺憾。
● 身體忠實地呈現弱點: 右轉失敗率高, 所以左半側從肩膀、手臂一路痛到膝蓋 Orz
● 搭臨時列車快速リゾートビューふるさと直達長野, 在姨捨站會關燈讓大家賞善光寺
平的夜景。


Day9 (1/6): 長野 → 野猿公苑、地獄谷溫泉

● 善光寺初詣 ... again。
● 搭電車、轉公車, 再走30分鐘山路, 到達野猿公苑。猴子們放鬆、放空的模樣, 實在
太療癒了。
● 秘湯後樂館, 溫泉水溫高、湯花多, 泡起來的感覺只有一個字, 爽!
● 目前為止, 我吃過兩次蚱蜢、一次馬肉, 都是在信州, 從這個角度看, 的確很符合「
未知を歩こう信州」這句口號 XD


Day10 (1/7): 野猿公苑 → 小布施 → 長野

● 清晨, 旅館簷下、露天池中都有猴群來襲, 在住客間引起一陣騷動, 赤裸裸地拍起照
來, 不亦樂乎 XD
● 猴子太有梗, 攝影者前撲後繼, 苑中有一個雪人, 右眼是 NIKON, 左眼是 TAMRON, 都
是大家留下的愛 :P
● 小布施途中下車。文化之町最有名的莫過栗子與北齋。
● 栗菓子の御三家, 吃完三分之二, 滿足。


Day11 (1/8): 長野 → 名古屋

● 對長野這個抬頭就可以看到(高)山的城市, 有股親切感。
● 從 0℃ 的長野回到 10℃ 的名古屋, 感覺很溫暖 :P
● 這次由飛驒、北陸, 途經北/中央/南阿爾卑斯、木曾路回到東海, 算是松本清張命案
路線, 高村薰的馬克斯之山也在這條線上 :P


Day12 (1/9): 名古屋 → 台北

● 上午 10:30 到達蓬萊軒松阪屋店, 前面已經排了上百人, 比三年前還過份 Orz
● 蓬萊軒裡的流程大概是這樣：入座 → 不假思索地點完餐 → 盯著別人桌上的食物 →
 上菜 / 食物照、自拍照、大合照 → 邊吃邊發出銷魂的呻吟 → 攤在座位上回味 → 付
錢走人。所以店內周轉很快 XD
● 中部空港開了一間「中部武將館」, 說是江戶時期的大名有八成都與中部有淵源, 但
店員竟打扮成一個紅色的外地(長野)人 :P
● 回到台北, 一開機立刻收到同事詢問何時上班的訊息, 提醒我美夢已經結束了 *_*
