去年很幸運買到了捷星特價票
本來以為4/12~17這樣的日期只能隨機應變看百名城的櫻吹雪
沒想到晚來的春天讓我們體驗了一趟名城花見

這次名城花見之旅的第一站是岡山城。
<a href="http://ppt.cc/W!XP" rel="nofollow" target="_blank">http://ppt.cc/W!XP</a>

因為他的牆壁是黑色的，所以又稱為烏城。
這座城相當老，在13XX年就開始興建，
目前除了再建的天守之外還有一棟月見櫓是重要文化財。

純文字版：

搭乘捷星六點五十分的班機，於十點半降落關西機場。

換了JR PASS後，按表操課，將行李放到姬路的旅館後，
便又搭乘新幹線前往一百多公里外的岡山

JR Pass的旅行，就是要新幹線搭到飽！

到達岡山站後我們要搭乘岡電巴士到後樂園去。
然後從後樂園逛到岡山城，再從岡山城另一頭坐路面電車回來岡山車站。

<a href="http://ppt.cc/2KlL" rel="nofollow" target="_blank">http://ppt.cc/2KlL</a>  岡山站出去，在四號月台搭乘公車在後樂園站下

下了巴士之後就是美麗的旭川畔，旁邊就是後樂園的入口。

後樂園與金澤兼六園還有水戶偕樂園並稱三大名園。來這邊一定要來看一下的呀！

後樂園是在1687年 奉岡山藩主池田綱政的命令動工建造的。至今具有三百多年的傳統。

庭園中非常遼闊，和兼六園給人的感覺不大相同
有很多日本新人在拍照
園中的建物為延養亭，供藩主造訪時家人的居所
可以一覽名園美景

不過這裡並不是賞櫻名所，只有邊邊有一排櫻花而已，有點失望。
<a href="http://ppt.cc/S5j5" rel="nofollow" target="_blank">http://ppt.cc/S5j5</a> <a href="http://ppt.cc/S5j5" rel="nofollow" target="_blank">http://ppt.cc/S5j5</a>

但是一下飛機就能看到這樣盛開的櫻花 真是讓人感到幸福無比


逛過後樂園，我們開始往岡山城走去。

岡山城是由數個海拔十多米的小山丘連結而建成
是梯郭式的平山城

<a href="http://ppt.cc/~85i" rel="nofollow" target="_blank">http://ppt.cc/~85i</a>

通往岡山城要走過這座跨過旭川的月見橋
旭川可以說是岡山城天然的護城河，保衛了整個岡山城的東面

過了河之後馬上就是岡山城的天守了

在明治時代，御殿，櫓，門等，大半被移走，後來天守也在二戰中焚毀
只剩月見櫓及西之丸手櫓存留至今

庭中有保留初代城主築城時的石垣，
看著這些穿越了數百年展示在我們面前的石頭，真是無限感慨。

現今所看到的門及天守，都是戰後重建的

而岡山城的天守是望樓型4重6階的建築
這樣子的造型在犬山城及丸岡城以及姬路城這些初期天守中也可以看到
同時期建造的天守其實不多
岡山城之外，就是会津若松城・松本城・広島城

本丸御殿已經不復存在，留下了一些石基
<a href="http://ppt.cc/oa_W" rel="nofollow" target="_blank">http://ppt.cc/oa_W</a>

天守的正面，入口就在正下方。
左邊白色的屋子是鹽藏 <a href="http://ppt.cc/bwBj" rel="nofollow" target="_blank">http://ppt.cc/bwBj</a>
這種在天守主建物旁邊又有小建物的天守稱之為複合式天守閣
這樣子的天守型式，一樣可以在犬山城，松江城以及小田原城中看到。

天守正面是一個千鳥破風一個唐破風一個入母屋破風
黑色的外牆加上添了金箔的瓦片，配色搶眼又華貴
因此別名是烏城或是金屋城
<a href="http://ppt.cc/!GAP" rel="nofollow" target="_blank">http://ppt.cc/!GAP</a>

天守內部都不能照相，不過大名駕籠可以照相。
古代日本人真的都很矮，我們要進去會很狼狽，因此就在外面拍照就好。
還有當時城下町商店的復元展示間可供小部份區域拍照

古時候的軍事設施，如今是民眾賞花野餐休憩的場所。
<a href="http://ppt.cc/4-2Q" rel="nofollow" target="_blank">http://ppt.cc/4-2Q</a>

城內可以看到的重要文化財之一便是月見櫓
<a href="http://ppt.cc/Y,BD" rel="nofollow" target="_blank">http://ppt.cc/Y,BD</a>
這也是岡山城唯二渡過戰災還屹立不搖的建物
月見櫓的軍事防衛功能不強，窗台會向外擴，主要給城主喝酒賞月之用。


離開了岡山城，我們往市區走一小段路去坐路面電車，
岡山市內的景色讓人感覺是很乾淨的一個地方。

最讓我們吃驚的是竟然在這邊看到小玉站長電車塗裝的陸面電車！！
這邊明明就跟貴志川線怎樣都打不著關係吧？！
<a href="http://ppt.cc/DQ4f" rel="nofollow" target="_blank">http://ppt.cc/DQ4f</a>

這是因為經營岡山電氣軌道和和歌山貴志電鐵的都是同一個集團
因此在2009年4月30日為了慶祝創立一百週年慶，
特別推出他們家的招牌貓到岡山來走動！

真是太Lucky~遇到了TAMA電車～
