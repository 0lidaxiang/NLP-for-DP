熟京都的人一定知道，京都車站上上下下到處都有美食名店，這間在京都車站 11 樓的
「名代豬排」就是其中之一。

<a href="http://www.flickr.com/photos/johnsonwang/6815184097/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815184097/</a>

門口這個人氣大蒸蛋看起來好美味，待會兒一定要來點一份：

<a href="http://www.flickr.com/photos/johnsonwang/6815183959/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815183959/</a>

也許是用餐時間剛過一點點，我們沒有遇到傳說中的排隊人龍，直接就順利入座了。店
內有中文菜單，別忘記要求拿來看，會輕鬆很多的：

<a href="http://www.flickr.com/photos/johnsonwang/6815182281/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815182281/</a>

這裡允許兩個人點一份餐喔！點好餐之後就送上這些器具，在台灣吃過日式豬排的人應
該就不陌生了：

<a href="http://www.flickr.com/photos/johnsonwang/6815182471/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815182471/</a>

桌上有三罐醬汁，其實都是豬排醬，隨客人喜好取用：

<a href="http://www.flickr.com/photos/johnsonwang/6815182619/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815182619/</a>

在吃吃喝喝，所以體驗一下味道就好。

<a href="http://www.flickr.com/photos/johnsonwang/6815182935/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815182935/</a>

白飯裝在比較大的容器內，再分食到小碗內方便用餐：

<a href="http://www.flickr.com/photos/johnsonwang/6815183073/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815183073/</a>

<a href="http://www.flickr.com/photos/johnsonwang/6815183233/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815183233/</a>

當然要看看豬排的斷面秀了：

<a href="http://www.flickr.com/photos/johnsonwang/6815183575/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815183575/</a>

可能是因為我們點 80g 的份量，所以豬排也沒辦法很厚，比起目前在心目中台灣最好的
日式豬排飯：杏子豬排來說（我們覺得勝博殿略輸給杏子），杏子豬排不但更厚更軟嫩，
肉汁也多。至於外皮的酥脆度就很好，只是好像也覺得與日本連鎖的和幸豬排，或台灣
的杏子豬排差不多就是。

整體來說，名代豬排沒有帶給我們很多驚喜，不知道是不是因為 80g 的米澤三元豬無法
充分表現名代的實力呢？

說真的，在台灣吃了杏子豬排十幾次後（當然杏子豬排是來自日本的技術），到日本再
吃連鎖的和幸豬排或京都名店的名代豬排，似乎沒有太多的驚艷之處，不免覺得若是準
備到日本吃這類炸豬排店，例如連鎖的和幸豬排的話，不妨在台灣多吃幾次杏子豬排（
當然這是針對離台北不遠的人說的啦…），到日本就把實力留給其他更多的美食吧！或
者要把炸豬排店的水準拉到更高的境界去挑選店家才會更值得些。另外也可能真的是杏
子豬排的技術移轉做得相當好，所以即使是在台灣就已經能夠吃到與日本當地差不多的
水準了吧！

人氣大蒸蛋是這樣：

<a href="http://www.flickr.com/photos/johnsonwang/6815183425/" rel="nofollow" target="_blank">http://www.flickr.com/photos/johnsonwang/6815183425/</a>

比對店門口的模型（請往前翻照片），實際上桌的樣子差異有點大。不過吃了幾口以後
就知道，其實模型要做那樣才能凸顯食材種類，實際做成蒸蛋後當然是沒辦法維持那樣
的外觀，不過用料很實在，模型有看到的食材都有出現在蒸蛋裡，而且蒸蛋很軟嫩，蒸
蛋的重要元素：高湯，也提升了蒸蛋的味道，相當值得一試。
