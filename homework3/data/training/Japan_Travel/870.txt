<span class="f3 hl">寫在遊記前</span>

謝謝版友們的經驗分享,謝謝尚華旅行社的詹小姐,謝謝小氣少年的詳細的網誌分享!!!

從一開始在版上、網上查資料,進而在版上問一堆蠢問題,最後到了日本的土地上旅行,

這趟自由行能成功並順利,真的要謝謝太多人的幫忙了!!!

日本真的是旅遊品質很好的國家,出發前才剛從大陸出差回來,

個人私心認為,大陸蘇州都遜於京都...

飲水思源,現在來分享一下旅遊行程,希望對其他版友有幫助!!!


<span class="f3 hl">5/1 (交通:飛機+JR=1320+160+160)</span>
<span class="f6 hl">原定行程:桃園機場&gt;&gt;&gt;&gt;関西空港&gt;&gt;&gt;&gt;新大阪地產飯店&gt;&gt;&gt;&gt;大阪梅田&gt;&gt;&gt;&gt;新大阪地產飯店</span>
                 日航      JR $1320          JR $160     JR $160

<span class="f2 hl">實際行程:同原行程</span>

梅田的百貨公司林立,可惜我對百貨公司沒啥興趣,再加上我一直孕吐非常不舒服,

基本上在梅田是走馬看花,連吃都隨便找間店吃一吃...

<span class="f3 hl">5/2 (交通:JR+京都巴士一日券$500=540+500+540)</span>
<span class="f6 hl">原定行程:新大阪地產飯店&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;金閣寺&gt;&gt;&gt;&gt;銀閣寺</span>
<span class="f6 hl">         &gt;&gt;&gt;&gt;清水寺&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;新大阪地產飯店</span>
<span class="f2 hl">實際行程:新大阪地產飯店&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;金閣寺&gt;&gt;&gt;&gt;銀閣寺</span>
                     JR $540       巴士      巴士

<span class="f2 hl">         &gt;&gt;&gt;&gt;清水寺&gt;&gt;&gt;&gt;祇園/花見小路&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;新大阪地產飯店</span>
         巴士      巴士             巴士      JR $540

日本人很守秩序,到了景點大家都依序跟著"順路"的指標緩緩前進,

拍照的時候,大家也會互相讓一下,雖然這天下著細雨,但行程相當順利,

離開清水寺大約傍晚,所以決定增加祇園這個點,是很豐富的一天~

<span class="f3 hl">5/3 (交通:JR PASS+京都巴士+奈良公車=5000/3+220+220+200)</span>
<span class="f6 hl">原定行程:新大阪地產飯店&gt;&gt;&gt;&gt;JR奈良站&gt;&gt;&gt;&gt;奈良公園/東大寺&gt;&gt;&gt;&gt;JR奈良站&gt;&gt;&gt;&gt;京都車站</span>
<span class="f6 hl">         &gt;&gt;&gt;&gt;二条城&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;JR難波&gt;&gt;&gt;&gt;新大阪地產飯店</span>
<span class="f2 hl">實際行程:新大阪地產飯店&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;二条城&gt;&gt;&gt;&gt;京都車站&gt;&gt;&gt;&gt;JR奈良站</span>
                      JR PASS      巴士      巴士       JR PASS

<span class="f2 hl">         &gt;&gt;&gt;&gt;奈良公園/春日大社/東大寺&gt;&gt;&gt;&gt;JR奈良站&gt;&gt;&gt;&gt;心齋橋/道頓堀</span>
         徒步                        巴士       JR PASS

<span class="f2 hl">         &gt;&gt;&gt;&gt;新大阪地產飯店</span>
         JR PASS

這天相較前一天,算是比較鬆的行程,也是玩的最開心的一天,

因為二条城一直是我老公想去的地方,奈良公園的小鹿是我一直想看的~

早上搭錯JR線,所以就將錯就錯先去二条城,再去奈良~

雖出了小差錯,但不影響行程,甚至給了奈良公園更多時候閒晃~

<span class="f3 hl">5/4 (交通:JR PASS=5000/3)</span>
<span class="f6 hl">原定行程:新大阪地產飯店&gt;&gt;&gt;&gt;嵐山&gt;&gt;&gt;&gt;森之宮/大阪城公園-天守閣</span>
<span class="f6 hl">         &gt;&gt;&gt;&gt;天王寺-通天閣/新世界&gt;&gt;&gt;&gt;新大阪地產飯店</span>
<span class="f2 hl">實際行程:新大阪地產飯店&gt;&gt;&gt;&gt;嵐山&gt;&gt;&gt;&gt;森之宮/大阪城公園-天守閣&gt;&gt;&gt;&gt;新大阪地產飯店</span>
                     JR PASS JR PASS                     JR PASS

在嵐山逛了好一會,去了天龍寺,渡月橋,嵐山大街,常寂光寺,野宮神社,嵯峨野竹林,

其中覺得走在竹林小徑是最舒服的!!

原本計劃去通天閣,但已經很累了,加上老公覺得看過大阪城公園-天守閣就足夠了,

決定回飯店休息...畢竟,我還是持續在孕吐啊...&lt;囧&gt;

<span class="f3 hl">5/5 (交通:JR PASS=5000/3)</span>
<span class="f6 hl">原定行程:無</span>

                     JR PASS       徒步              JR PASS

其實這天我是沒安排的,打算想去哪就去哪,所以臨時決定去神戶走走,

行李箱就放JR新大阪站內的寄物箱,05:00-24:00內600日元,

雖說不便宜,但換得旅行方便~

一到舞子站就聞到海的味道,這趟旅程算是上山又下海了~


<span class="f3 hl">寫在遊記後</span>

因為從沒去過日本,也不熟悉日本関西的交通系統,所以交通上選擇JR PASS,

如果選用KTP,相信在旅費上會更省錢,KTP還含蓋部份景點的優惠及京都巴士,

可惜對初心者的我們來說,相對複雜~

下次日本旅行,會考慮跟團比較看看旅行的品質是否會提高,

我這種愛操心又急性的人,自由行雖然彈性,但卻時時刻刻在控管預算和跟著行程跑,

其實心情是非常緊繃的...

無論如何,這是一趟經驗寶貴又順利的旅行:)
