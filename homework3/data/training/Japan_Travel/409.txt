ほうとう 是一種山梨縣（故甲斐國）的鄉土料理，由扁平的烏龍麵加上
蔬菜及味噌燉煮而成的一種麵食。(截自Wiki)

河口湖雖然是觀光的熱門區域，但是經營吃的商家卻不多，許多甚至晚上
五六點就關門了。

這也難怪，多數人來到河口湖都是搭配旅館一泊二食或者當日來回，要不
就是從箱根或富士吉田過來，這也限制了這邊的發展。

ほうとう不動是一家在河口湖有三間分店與一間茶屋的餐廳，分別是河口
湖北本店、河口湖南店、河口湖駅前店與不動茶屋。這次我們選的是河口
湖北本店 ほうとう不動。
<a href="http://tinyurl.com/7j95rsj" rel="nofollow" target="_blank">http://tinyurl.com/7j95rsj</a>

會來這邊吃是因為接下來的行程會到河口湖木ノ花美術館，而到 ほうとう
不動消費的話可拿門票割引券，讓木ノ花美術館的門票只要200円，而一般
的割引券則是打九折(500円-&gt;450円)。另一方面也是因為沒有太多選擇的關係。
<a href="http://tinyurl.com/7oqp4hb" rel="nofollow" target="_blank">http://tinyurl.com/7oqp4hb</a>

ほうとう不動 的菜單很簡單，招牌就是不動ほうとう，另外還有
馬刺身(生馬肉)、豆皮壽司等，二話不說我們就點了不動ほうとう。
<a href="http://tinyurl.com/7mqyz8x" rel="nofollow" target="_blank">http://tinyurl.com/7mqyz8x</a>

由於我們11點就到店了，是店內的前幾批客人，店家剛開門營業第一批上菜
的時間總是需要比較久一點，我們就稍微逛了一下店內。

河口湖北本店 ほうとう不動 是一間木造建築，店內很大，有分為椅子席與
榻榻米席。
<a href="http://tinyurl.com/7mlklrs" rel="nofollow" target="_blank">http://tinyurl.com/7mlklrs</a>
<a href="http://tinyurl.com/7jjcp3t" rel="nofollow" target="_blank">http://tinyurl.com/7jjcp3t</a>

這裡營造的是一種過去的氣氛。
<a href="http://tinyurl.com/7k8g73l" rel="nofollow" target="_blank">http://tinyurl.com/7k8g73l</a>
<a href="http://tinyurl.com/7ygzgae" rel="nofollow" target="_blank">http://tinyurl.com/7ygzgae</a>

這裡的杯子也非常有特色。
<a href="http://tinyurl.com/7aj2jqv" rel="nofollow" target="_blank">http://tinyurl.com/7aj2jqv</a>
桌上擺著的鐵器是等下上不動ほうとう的時候隔熱用的。
<a href="http://tinyurl.com/8xyxuxb" rel="nofollow" target="_blank">http://tinyurl.com/8xyxuxb</a>


等候多時的不動ほうとう終於上菜了，熱騰騰的裝在鐵鍋裡送上來，
店家還特別交代鍋子很燙，要避免碰到鍋子。
<a href="http://tinyurl.com/85da4cu" rel="nofollow" target="_blank">http://tinyurl.com/85da4cu</a>

ほうとう 果然如同Wiki上寫的是由扁平的烏龍麵加上蔬菜及味噌燉
煮而成的一種麵食，吃起來除了有些鹹外，其實還蠻健康的，因為都
是蔬菜，有大白菜、菇類、南瓜等。麵條吃起來蠻Q 的，不過實在太
多，咬到後來覺得好累。

他給我的感覺吃起來非常的鄉土，給我的感受就如同第一次去秋田吃
到米棒鍋一樣，一生要來吃一次，但應該不會再來吃第二次。

這幾年到日本吃東西有一個心得，只要是地區的名物、鄉土料理，而
且在其他區域不容易找到，那這料理就是屬於那種會吸引某些人的口
味，但往往我就是不愛。想一想也是，如果這些料理是非常大眾的口
味，應該不會只侷限在地區的鄉土料理。(喂!有人這樣寫食記的嗎?)
也許這些料理比較適合日本人口味吧。


河口湖北本店 ほうとう不動 的位置在湖的北岸，可以搭乘遊湖巴士
在河口湖オルゴールの森 下車，然後沿著一條小路走一下就到了，
沿路都有明確的招牌指示。另外在河口湖駅站前還有一間分店，用餐
都可以取得 河口湖木ノ花美術館的割引券。

店名: ほうとう不動 河口湖北本店
