</span><span class="hl">PAUL&amp;JOE 2012貓咪春妝(眾所期盼)/開始缺貨了...</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743935&amp;p=138" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743935&amp;p=138</a>

一直到這幾天，才有時間去櫃上結已經先預購的PAUL&amp;JOE 2012貓咪春妝，來到櫃上，
PAUL&amp;JOE生意還真是相當好...而且還是平常日喔，只能說，貓咪的春妝實在是吸引力太
強了。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743936&amp;p=139" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743936&amp;p=139</a>

限量的貓咪春妝也終於開始陳列在櫃上，這次的貓咪春妝PAUL&amp;JOE配合好多的彩妝活動，
無論是結合玉山信用卡，或是結合相關貓物、愛貓合照等都享有特別的小禮物。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743937&amp;p=140" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743937&amp;p=140</a>

也好險有先預購，星期五去的時候，櫃姐說蜜粉餅、腮紅棒甜美粉002都已經沒貨了...，
不過我這次自己買的是不多，單純有些只是覺得可能漂亮和值得收藏才買來留著，拍拍照
，寫寫文章，當作藝術品欣賞這樣...大概就要開始變成所謂的「供品彩妝」了。

但我自己覺得，這一系列，如果要論實用度，大概就屬眼影和指甲油最值得購買，但真得
要談論到可愛度，就外型上，還是蜜粉餅、腮紅棒大勝了...。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743938&amp;p=141" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743938&amp;p=141</a>

這是消費就有送的貓咪便條紙，頗可愛的。




<span class="b1 hl">貓貓眼彩盤</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743940&amp;p=143" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743940&amp;p=143</a>

這是這次的眼影盤，很喜歡他的外盒設計，剪影式的貓咪，旁邊還畫有隻小鳥，根據不同
的顏色，也有不同的花紋設計，重點是他12g竟然只要1400元...算是PAUL&amp;JOE中相當佛心
的商品...。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743941&amp;p=144" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743941&amp;p=144</a>

打開來主要是鬆緊帶式的紙盒包裝，然而，這樣的紙盒包裝實在不適合隨身攜帶，一來容
易髒，二來也容易傷到紙盒...漸漸地，他就真的會越來越醜了...。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743942&amp;p=145" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743942&amp;p=145</a>

我自己帶的076優雅暹羅貓，標準的大地色咖啡色系眼影，就實用度來說，也真的是比較
實在一些。但真的去細看眼影盒，它的設計就比較沒有其他產品來的那麼可愛了...。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743943&amp;p=146" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743943&amp;p=146</a>

其中，我特別喜歡他左邊那個古銅金色...，實在是太好看了，而且12g時再適用到天荒地
老啊...，再加上整個眼影粉帶有PAUL&amp;JOE家經典的橙花香，邊上都覺得也太香了，整個
完妝如果都是用PAUL&amp;JOE的產品，應該整個臉都是香噴噴了...。



<span class="b1 hl">宮廷小貓修容棒</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743944&amp;p=147" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743944&amp;p=147</a>

接著是眾所注目的腮紅棒。外型的設計上，紙盒倒是挺特別的，4g訂價是1200元。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743945&amp;p=148" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743945&amp;p=148</a>

包裝的設計上就是一般PAUL&amp;JOE的口紅設計，所以當時在國外釋出相關消息的時候，當下
都以為這是唇膏，畢竟之前PAUL&amp;JOE確實也是有推出貓頭的口紅...。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743946&amp;p=149" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743946&amp;p=149</a>

開蓋後，在這邊就有印上限量貓咪圖案的設計。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743947&amp;p=150" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743947&amp;p=150</a>

傳說中的戴著禮帽，打著領結的貓咪腮紅棒...，這麼可愛，到底叫人該怎麼用啊。而且
由於是到櫃上，可以直接試，我都會覺得這系列的腮紅棒實在不是很顯色，而腮紅棒的使
用上又真的有難度，就覺得就還是帶個002甜美粉就好了，結果當然也是這個色號最搶手
...。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743948&amp;p=151" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743948&amp;p=151</a>

櫃姐是說，基本上就從後腦勺開始用，像貓咪一樣畫三撇鬍鬚在臉上，然後再暈開來，就
會呈現很自然的腮紅色系。...但你說，你捨得用嗎。




<span class="b1 hl">貓貓唇彩</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743949&amp;p=152" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743949&amp;p=152</a>

接著是口紅，同樣是剪影式的紙盒設計，就整個外觀上來看，實在是相當可愛。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743950&amp;p=153" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743950&amp;p=153</a>

打開來，口紅的設計上也是以紙為主，就比較典雅，比較日系，當然也就是很PAUL&amp;JOE。
只是每每我都覺得PAUL&amp;JOE如果用這樣的紙式設計，實在是很難保存...，容易髒，又容
易傷。

<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743951&amp;p=154" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743951&amp;p=154</a>

口紅上共有十個腳印，象徵PAUL&amp;JOE彩妝的10周年。只是我會覺得這系列的口紅竟出乎意
料的是作霧面的...還真是有些不討喜，因為感覺有些乾，也很霧，使用上就真的比較挑
人用了。



<span class="b1 hl">貓咪指甲油</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743952&amp;p=155" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743952&amp;p=155</a>

最後是指甲油，其實這次貓咪三色的指甲油都還蠻不錯的，但可惜就可惜在整個設計只有
外紙盒很可愛...，打開來...

<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743953&amp;p=156" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743953&amp;p=156</a>

就是很一般的PAUL&amp;JOE指甲油瓶身設計，相對上，就比較輸給其他產品的注目度，不過這
次這三色指甲油倒是都還挺不錯的。

<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743954&amp;p=157" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743954&amp;p=157</a>

我只帶一罐025 Baby Cat，裸膚色的指甲，就好像剛出生的小貓咪一樣的貓爪乾淨。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743939&amp;p=142" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963743939&amp;p=142</a>

整個2012 PAUL&amp;JOE貓咪春妝大概就以上，結果看到DM，竟發現還有個回馬槍...就是在春
季化妝品節消費滿3500元，送貓咪化妝包...這有點可愛的太犯規了...，但偏偏是要在春
季化妝品節檔期才可以，現在消費達到也是沒有送，但這些限量貓咪彩妝，如果再不趕快
搶，最好等到春季化妝品節還會有啦...，蜜粉餅、腮紅棒都開始缺貨了...，真的，要快
點去專櫃訂了，不然就真的都要沒了。

不過我之所以沒有買蜜粉餅的原因，一來是覺得有點太貴之外，主要也是我只喜歡那個貓
掌粉撲...外盒過於塑膠感實在不是我的菜。
