</span><span class="hl">CHANEL 絲絨珠光蜜粉/限量的美好</span>
<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752783&amp;p=228" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752783&amp;p=228</a>

大概12月底，1月開始，蜜粉用的差不多後，就一直在找尋新的蜜粉，我在挑尋蜜粉的前
提還是會希望就g數和價格換算下來，不要太驚人，所已找了一會兒，最後在1月的時後，
趁著板橋大遠百的開幕，在CHANEL有95折的狀況下，購入了這款去年9月出的限量絲絨珠
光蜜粉，因為他的色澤實在是太美了。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752784&amp;p=229" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752784&amp;p=229</a>

一直都還沒有機會整理一下這塊蜜粉的照片和心得，當有人問我有什麼蜜粉好用時，我都
會先說我有買一款CHANEL去年下半年出的有帶珠光的蜜粉很好用之類，但很多人就會困惑
，都會連想到聖誕彩妝的那顆蜜粉"精靈"，但當然我不是在說那款，因為那款蜜粉亮粉實
在閃得嚇人，連櫃姐都說只適合拿來打身體。現在我才把他正名是限量的絲絨珠光蜜粉，
雖然說是限量的，但櫃上可能還問的到，只是量很少。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752785&amp;p=230" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752785&amp;p=230</a>

基本上，雖然這款蜜粉是叫絲絨珠光蜜粉，但他整體其實是Follw輕盈完美蜜粉，同樣也
是30g，價格是1800元，不過今年已經漲到1840元了。不過以這樣的價格，這樣的容量，
的確算是CHANEL家中相對便宜的產品。其中，他的蜜粉撲就是比較一般的棉撲，真的要和
今年2月推出的珍珠光感淨白防曬蜜粉的天鵝絨粉撲比起來，還是有所落差。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752786&amp;p=231" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752786&amp;p=231</a>

全新的蜜粉鬆粉，基本上都會有塑膠貼膜，通常會建議在撕的過程不要全撕掉，不然有時
後容易倒出太多的蜜粉，但如果你本來就不會帶著它趴趴走，全撕掉就也無妨。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752787&amp;p=232" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752787&amp;p=232</a>

而撕掉後，大概就是一些正方型小格，因為這款係為絲絨珠光蜜粉 ，所以單這樣看，就
蠻能看得出它其中的光澤感，不過這樣的絲絨珠光蜜粉中，就沒有再多一層濾網去篩，但
也還好這款絲絨珠光蜜粉真的是在粉質上還蠻細的。


<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752788&amp;p=233" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752788&amp;p=233</a>

倒出一點蜜粉在手上，可以見到其蜜粉的顏色是粉膚色調的，也不會太有白色過白的困擾
，或者是太深的困擾，他就大概比一般粉底的顏色稍微再淺一些，不過單這樣去看它的顏
色或質地，也還不是太準，等到一畫開後，特別就很明顯。



<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752789&amp;p=234" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752789&amp;p=234</a>

用半邊半邊的方式去對照，當你把蜜粉推開後，其馬上化為絲絨般的珠光質感，在還沒完
全畫勻前，容易顯白，但在完全畫勻後，其呈現出的光采妝感，也就完成了。




<a href="http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752790&amp;p=235" rel="nofollow" target="_blank">http://www.wretch.cc/album/show.php?i=Goris&amp;b=20&amp;f=1963752790&amp;p=235</a>

這就是大概格刷勻後的蜜粉妝感，因為透過近拍的關係，他的整個光澤感就顯得更明顯，
基本上，他的妝容感，就是如同他的名稱一樣，是帶有珍珠絲絨感，那種緞面緞面，卻又
透出細緻珠光的蜜粉妝感，在上蜜粉的同時，同時提亮臉部亮度，煥發出光采。

在使用這款絲絨珠光蜜粉時，我還是會拿來上全臉，他看似珠光明顯，但真的上在臉上後
，轉換而成的是臉部的光澤感，這樣的光澤感，很快地就能掃除掉有時後氣色欠佳的暗沉
感，又或者是粉底過深，甚至是太霧面的粉底妝容，透過這個絲絨珠光蜜粉，有點像立即
為你的臉罩上一層紗，這個紗的光澤卻是透過光線的折射和角度的不同去顯露出光澤感，
卻又不致於是到金光閃閃的感覺，而是照上後，整個臉色是明亮、有神的，後來我也都把
他當作急救型蜜粉，立即改善臉色的感覺。但如果你真的要說他有什麼效果，除了妝容上
，保濕、控油等那些就比較沒感覺，就比較是基本的妝容蜜粉了。

基本上，這款蜜粉櫃上應該都還問的倒，只是貨量應該就是不太多，因為大概是去年9月
推出，我一直到年初的1月還在板橋大遠百用95折買到，2月去買純淨粉蓮時，和櫃姐聊到
這款蜜粉很好用，他也有提到櫃上尚還有一瓶，大概就是這樣的貨量吧，給大家參考。
