還記得上次寫這主題，是2008年，之後就年年心有餘而力不足.......<span class="f0 hl">(汗)</span>
今年努力點，終於生出來了

話不多說，直接切入主題。


<span class="f3 b2 hl">妝前防曬 - ALLIE</span>

依舊由Kanebo Allie蟬聯好幾年。

坦白說，真的是不錯用，個人認為美白款較清爽，不過每個人的感覺都不太一樣，
唯一的缺點大概就是「<span class="f6 hl">年年改款，年年漲價</span>」，叫消費者情何以堪阿～～～～

但我還是年年買了orz....

雖然每年都希望是不是可以改用安耐曬，但還是都固定購入Allie，
今年，我一定要來用看看安耐曬，以了心願。

恭喜妳 ，Allie小姐，今年度又蟬聯了呢，辛苦妳了。


<span class="f3 b2 hl">粉底 - MUF雙用水粉霜</span>

在MAC柔霧無暇整罐用完吃光光後，
購入了傳說中各大部落客及各大知名美妝網站評選都佔有超高比例高分數的水粉霜。

從MAC柔霧切換過來，老實說起初是不太適應，
因為柔霧擁有很厚重的妝感及強大的遮瑕能力，
比起MUF水粉霜的輕透光亮卻沒啥遮瑕程度的妝感，差別很大。

漸漸的適應習慣後，<span class="f6 hl">發現雖然MUF水粉霜無法同MAC柔霧有超強遮瑕力，</span>
<span class="f6 hl">但卻能將臉上的泛紅及不平均的膚色掩飾的非常的好，</span>
<span class="f6 hl">在出油以後更能夠表現其極佳的妝感，讓臉部肌膚看起來水嫩水嫩。</span>

所以，2011年度最佳底妝，就送給呢MUF水粉霜，恭喜妳阿！

在水粉霜即將用完的同時，購入了肌膚之鑰光緞，希望今年妳也別讓我失望阿。


<span class="f3 b2 hl">遮瑕 - MUF緊緻遮瑕霜</span>
<span class="f6 hl">遮曬斑、遮小痔、遮痘痘、遮黑眼圈，</span>
<span class="f6 hl">大遮小遮都靠它。</span>

基本上，我的萬年過敏黑眼圈我不奢求有任何遮瑕產品可以拯救它，
畢竟那是要靠體質上的調理才有辦法去除。

所以遮瑕霜如果能夠幫我遮掉50%，我就已經很感到慶幸了。

我沒辦法宣稱說，這條MUF的遮瑕霜有多好多好，
但至少他能滿足我遮東遮西的基本需求。

重點是又不乾，因為質地偏水，所以一整天下來也不會龜裂，
讓整個臉部肌膚看起來乾乾的，一乾，就老了。

能夠遮掉整個臉部80%的缺陷，我就認為它是一條好的遮瑕膏。

目前使用過的遮瑕膏，它算是數一數二好的了，
私心認為更好的是，是肌膚之鑰，可是太貴了，等姐姐有錢再去把妳帶回家嘿，等我！

MUF遮瑕霜，C/P值超高，超大條用到天荒地老，
如果妳跟我一樣要求不多，對不完美深信不疑，衷心推薦妳，它會是你的好朋友。


<span class="f3 b2 hl">修容 - MAC柔光炫彩餅</span>

手頭上有MAC產品的人，對這塊大餅一定不陌生，
是的，她也是連續蟬聯了好幾年<span class="f0 hl">（因為好用，重點是用不完）。</span>

10G的容量實在是叫人又愛又恨，
對於大餅臉的我，實在是超級好的福音，但每天不管怎麼用，就是毫無消瘦之意。

如今，終於凹陷了，感謝老天爺，
還見到傳說中的鐵片軍了呢！

原本圓滾滾的她，如今凹了個缺，表面超凹的，
上面那張看不清楚沒關係，我們看下面這張

而且還不小心被發現是右撇子了，因為刷子刷的方向，
凹陷的都是左半部。

再給我一點時間，我想，很快就會用光的，
恩......很快。


<span class="f3 b2 hl">修容(打亮) - CEZANNE明亮修容組</span>

剛剛MAC的大餅，是修小臉的，CEZNNA是打亮的，
兩個不曉得可不可以算在同一個項目<span class="f0 hl">(汗)。</span>

開架裡面的好物，看到鐵片軍就曉得有多愛她了
<span class="f0 hl">（坦白說，我不曉得現在還買不買得到，因為購入年代已久）。</span>

當初只是因為資生堂的四色修容打亮格用光了，
又不想要買超大塊的MAC大餅來打亮，於是繞了一圈屈臣氏就帶回了她。

便一直使用到現在，我想，今年底可能就壽終正寢了，
又要尋覓新的打亮了......

如果有推薦的口袋名單，歡迎一併推薦給我阿


<span class="f3 b2 hl">腮紅 - MAC FAFI 限量腮紅</span>

我一直在猶豫，究竟年度腮紅是要頒給MAC FAFI這顆，
還是要頒給GIORGIO ARMANI的3號。

畢竟自己使用率最高的還是GIORGIO ARMANI，
但最後為什麼頒給了MAC呢。

<span class="hl">我想決勝點應該是在「實用搭配性」這點。</span>

GA 3號其實真的很挑膚色跟挑妝，雖然每次上都很好看，
但是百搭性其實不佳。

但我必須澄清的是，GA無論粉質 / 顯色度 / 質感，都是大勝MAC的。

所以下次如果購入了其他顏色，
那我想獲勝的就非GA無誤了。

NARS腮紅雖然也有，但沒想到，去年一整年，
我居然完全沒有使用過......(驚)



<span class="f3 b2 hl">眼部打底 - UDPP</span>

她的好，不用我多說了，
隨便打個關鍵字「UDPP」，美妝版 / GOOGLE / YOUTUBE，
都可以搜尋到超級多的文章。

還有各種不同顏色款式提供大家選擇，
硬管軟管皆有。

唯一的缺點大概就是購入不易。

所以在她即將用完的同時，
我已購入NARS眼影打底筆<span class="f0 hl">（還好在漲價前購入）。</span>

今年，就拜託妳們兩位拉。



<span class="f3 b2 hl">染眉膏 - Fasio</span>

開架中的染眉好物。

其實一直想購入MAC的染眉膏，因為每次看到SHOCKISGOOD的文章，
都繪有鼓衝動想殺去MAC櫃直接購入。

但轉頭看看自己的彩妝櫃，靜靜躺在那邊的FASIO，
那衝動的心情，瞬間被澆熄。

年紀大了沒有每天化妝，
還是乖乖的把一樣東西用完再買下一樣吧......

記得，定價似乎是170，不是很確定是否有新改款，
但是效果真的不錯，只要有化妝一定會使用到它。



<span class="f3 b2 hl">睫毛膏 - INTEGRATE根根飛揚睫毛膏</span>

有陣子，非常熱愛貼假睫毛，
因為覺得自己的睫毛雖然很長，可是很軟，而且當時又追求很誇張的眼妝，
所以對假睫毛熱愛無比。

現在已經不是非常著重眼妝，
所以幾乎都擦睫毛膏。

根根飛揚可以讓睫毛有很自然的妝感，但又可以有畫龍點睛的效果，
不會過度張揚，是一種隱藏的美。

在我心中，她已經打敗kissme了



<span class="f3 b2 hl">眼影 - LUNASOL</span>
<span class="f6 hl">LUNASOL榮登去年度眼影王無誤。</span>

雖然，永遠搞不懂，這些大地色系在肉眼看來完全一模一樣，
怎麼可以每季都出大地色，卻還是每季都狂熱賣。

但每每使用於眼皮上，
便會了解其中奧妙之處。

LUNASOL的好，便在於光澤度的顯現，
提升整體眼妝的質感，
讓妳從勤儉刻苦的小資女瞬間轉換成雍容華貴的大貴婦。

加分，加分，大加分啊！！



<span class="f3 b2 hl">唇膏 - LUNASOL</span>

自從購入了LUNASOL的唇膏。

CHANEL掰掰；
DIOR掰掰；
MAC掰掰；

說真的，目前還沒有找到比LUNASOL更滋潤的唇膏，
資生堂我不熟，沒有用過，所以無法評論。

但是歐美各大櫃，我都沒有找到更滋潤的。

而且，用了它之後，我都不用唇蜜了，
因為已經顯現了水潤度，就不需要再額外使用唇蜜了。

除非當日想要超誇張唇妝，才會額外使用唇蜜。

謝謝妳，超滋潤的LUNASOL。


<span class="f1 hl">以上，為2011年度十大彩妝。</span>
