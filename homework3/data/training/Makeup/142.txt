戀愛魔鏡 - 睫毛王 使用心得

最近這支睫毛膏很多人討論
當然是因為他的名字實在是太囂張了
不說廢話了
睫毛王的刷頭還蠻大的
纖維非常之多
多到打開來的時候 刷頭前面會沾黏一堆的睫毛膏
很濃稠 要在瓶口ㄍㄡˊ個幾回才能甩掉
( 圖 )
還沒上睫毛膏的照片
( 圖 )
上了一層睫毛膏的照片
(有先用睫毛夾將睫毛夾翹再上睫毛膏)
( 圖 )
因為纖維的量很多 一下子就刷長了 也很翹
半乾之後在多上兩三層都還上的去
會越刷越濃 也會再長一點點
有網友覺得纖維太多會變蟑螂腳
我自己覺得還好 如果有睫毛梳的話
應該梳一下就能解決蟑螂腳的問題
由於我戴假睫毛都會眼睛不舒服
可是我又非常喜歡又長又濃的睫毛妝感
所以我都會刷很多層 (兩三層)
我覺得這支睫毛王還不錯
可以刷得又長又濃
缺點是 刷頭太大 纖維太多
刷下睫毛不是很好刷
所以如果平常都戴假睫毛
只刷下睫毛的女孩

""不要買MJ睫毛王喔!!""

我的下睫毛是用這隻FASIO刷的 (名字我也忘了)
( 圖 )
我覺得下睫毛還是要用這種很像梳子型的比較好刷!
然後這是畫好的眼妝!!
我覺得上睫毛用睫毛王畫出來的效果很不錯!!
( 圖 )
還沒結束!!
畫完這個妝之後
我去外面剪了頭髮晃了一圈
五個小時後回家
眼妝沒甚麼暈
睫毛的翹度沒有剛畫好的時候那麼翹了
不過我覺得還是很翹
( 圖 )
個人很滿意睫毛王的表現
如果化眼妝跟我一樣 想要很濃很翹 不常戴假睫毛的
推薦啦!!!
