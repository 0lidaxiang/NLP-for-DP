以下為複製貼上的bbs版



第一次要分享自己的年度彩妝，有莫名的感覺

大概是很怕自己喜歡使用的別人覺得是地雷吧！不過每人膚質狀況都不同

如果覺得難用拜託不要罵我....

我自己覺得膚質是混合偏乾，T字會出油但不大，臉頰則是乾乾的

這次要介紹的東西都是平價品，大概睫毛膏是最貴的

那就來一一介紹吧！

<a href="http://wenshing.pixnet.net/album/photo/189816934" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816934</a>

先來看看全部產品！但我少放了粉餅，因為借人用了

朋友要找適合他的底妝產品，我就把我有的都借她實驗囉！

不過這樣看起來產品還真的很少耶！



那就先從底妝介紹起

<span class="f6 hl">隔離霜</span>
<a href="http://wenshing.pixnet.net/album/photo/189780794" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780794</a>
<span class="f3 hl">Orbis隔離霜（藍蓋）</span>

我喜歡這條Orbis隔離霜，很多人說滋潤型不好用，但我很喜歡

當然，我也有清爽型！會視情況看要選擇哪條使用

現在天氣比較冷，我就會選擇滋潤型的，夏天則是使用清爽型

<a href="http://wenshing.pixnet.net/album/photo/189780798" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780798</a>

朋友說不喜歡滋潤型的質地，但我覺得還好呀！

平常我會擠一坨在手上，沾取之後直接塗抹，這次則是要拍照才先點在臉上

後來查詢之後好像有人說要快速推開才會效果比較好

大概是我每次都是很快推開所以覺得它很適合我吧！

新款的我還沒有用過，之前趁要改版前的出清特價，一次都買2組共4條(當時有滿千折百)

折起來一瓶只要245元！

但除了現在服役中的滋潤型，只剩下一罐清爽型了

查了Urcosme似乎新款的滋潤型還不錯，之後手邊用完會考慮購入！



<span class="f6 hl">遮瑕膏</span>
<a href="http://wenshing.pixnet.net/album/photo/189816254" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816254</a>
<span class="f3 hl">MAYBELLINE  純淨礦物遮瑕筆</span>

這支遮瑕筆我覺得還不錯耶！之前媚比琳大特價的時候買起來真的很划算

原價290元，如果打完85折再兩件折價，價格很可愛

雖然效果我覺得普普，無法完全遮乾淨，不過也算是可以了！

<a href="http://wenshing.pixnet.net/album/photo/189816256" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816256</a>

他在遮黑眼圈方面效果還好

左上是上完Orbis隔離霜之後的樣子，右上則是遮瑕後再蓋上粉餅

在遮痘痘方面還可以，應該很明顯哪個有遮過吧！

而且棒狀設計整個很好控制，稍微點一點再推開就好

<a href="http://wenshing.pixnet.net/album/photo/189816262" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816262</a>

雖然遮得不是很乾淨，但跟沒遮得比起來真的有差有差有差

<a href="http://wenshing.pixnet.net/album/photo/189816264" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816264</a>

跟手邊的另一個遮瑕比一比，比較保濕也比較自然

BeautyMaker遮瑕的部分為左眼（右邊）

當然有可能是因為BeautyMaker我挑錯顏色了，當時特價199元只有粉膚色就買了

原本想要趁特價的時候再購入健康色來調色，後來作罷，因為我覺得有點乾

目前maybelline是我的新歡！



<span class="f6 hl">粉餅</span>
<a href="http://wenshing.pixnet.net/album/photo/171194217" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/171194217</a>
<span class="f3 hl">BeautyMaker傳明酸美白防曬柔膚粉餅</span>

目前為止喜歡的粉餅則是這款傳明酸粉餅

這原價不便宜，但常常特價就是，我是趁299元的時候購入的！

雖然他在各方面都不是非常優秀，但妝效我自己挺愛的

平常如果上粉底液我甚至會使用蜜粉刷直接當刷這塊當蜜粉

如果上完隔離霜要直接上這塊，就會使用裡面附的粉撲

嚴格來說這塊遮瑕度不高，但因為這樣我才會偶爾拿來當蜜粉使用

（平常太懶常常只上粉餅就完成底妝啦！偶爾才會上上粉底液！）



額外推薦：較平價的－CEZANNE、ZA；較高價的－L'oreal光譜、Nature&amp;Co

(Nature&amp;Co之前差點手滑，因為粉盒好可愛！但就算有打79折價格還是沒有很可愛就是)


底妝就這樣非常快速的結束

開始其他部份了！



<span class="f6 hl">眉毛</span>
<a href="http://wenshing.pixnet.net/album/photo/189780828" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780828</a>
<span class="f3 hl">惹我　眉筆</span>

其實我不太常畫眉毛（自以為其實算有眉毛而且有瀏海就忽略）

但有天堂弟跟我說我眉毛很淡，畫一下會比較好

所以就買了這便宜又好用的惹我眉筆，效果挺不錯挺自然的，我買的是深灰色



<span class="f6 hl">腮紅</span>
<a href="http://wenshing.pixnet.net/album/photo/189781370" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189781370</a>
<span class="f3 hl">media  柔光立體修容餅</span>

（上圖有打錯產牌名稱&amp;有色差）

平價的腮紅我很推薦這款！

就算是原價也是可愛的價格210元，又常搭配85折，更不用說之前寶雅有特價150幾元

我買的是RS2、BE2

它一共有10色，應該是可以挑到自己喜歡的吧

<a href="http://wenshing.pixnet.net/album/photo/189780782" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780782</a>

比較常用的是這款RS2，帶紫色調感覺會比較顯白

但照片很難拍出顏色就是，這塊平常會放在化妝包，所以整個看起來就舊舊髒髒的

<a href="http://wenshing.pixnet.net/album/photo/189780778" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780778</a>

這是BE2的效果，給人很有活力的感覺

會挑選這款，不只是價格可愛，還攜帶方便，內附的小刷子也不會太難用

而且這款我覺得不太會失手唷！算是適合新手使用吧！



<span class="f6 hl">唇蜜</span>
<a href="http://wenshing.pixnet.net/album/photo/189780772" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780772</a>
<span class="f3 hl">INTEGRATE  超完美艷光唇蜜</span>

我愛買唇彩，可是又很少擦

如果要選的話，我應該會推這款INTEGRATE的唇蜜

旋轉加上這種刷頭我覺得很好用，缺點大概是容量不多（但對我來說已經足夠）

黏度不太黏，不小心吃到也沒有什麼味道

朋友也曾經生火跑去買了另一個顏色，只是她說容易掉，但剛上上去很美

我買的色號是PK333跟RD721，較常用的是上面這款PK333



遺珠：essence極限光裸色唇蜜，這款c/p值更高，只是我不喜歡吃到的味道！



<span class="f6 hl">眼影</span>
<a href="http://wenshing.pixnet.net/album/photo/189780758" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780758</a>
<span class="f3 hl">INTEGRATE  黑瞳晶綻眼影盒</span>

眼影還是要推一下INTEGRATE的，這款我覺得很實用！

後面出的大概是色號買錯，反而很少用

比起也很優秀的戀愛魔鏡，這款是平常不知道要用什麼時會自動拿起來的

又常搭配85折，算是整體比較划算！

<a href="http://wenshing.pixnet.net/album/photo/189780760" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780760</a>

我買閃耀奶茶跟珍貴粉鑽，但比較愛的是閃耀奶茶啦！比較不容易出錯

<a href="http://wenshing.pixnet.net/album/photo/189780764" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780764</a>

很弱的試色照，眼線色比珍貴粉鑽淺，但也因為這樣覺得比較不會失手

平常也都是四色都會擦上去

<a href="http://wenshing.pixnet.net/album/photo/189780768" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780768</a>

本身眼睛條件不好，如果要有神就要暈很高，此時我比較會用眼影代替眼線筆

上這款眼影也沒有什麼技巧，就是疊上去而已

但要濃要淡可自己決定，平常只有像上圖這樣，下眼線想要閃亮也可以使用打亮色就好

這款平價又實用，我很喜歡！



額外推薦：Lavshuca的戀法迷棕眼彩盒評價很不錯，但一直沒買來嘗試

（眼影好幾盒根本用不完，害我不敢下手買新的！）



<span class="f6 hl">眼線</span>
<a href="http://wenshing.pixnet.net/album/photo/189780804" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780804</a>
<span class="f3 hl">Solone  30秒定妝慕絲眼線眼彩筆</span>

眼線大概絕對是Solone這款

我很少買眼線產品，太習慣用眼影來代替了，但硬要找眼線產品會是它得獎

這款可以當眼線也能暈開當眼影

我買了三種顏色：森林綠、濃咖啡、香檳粉

濃咖啡目前還沒用過，但評價很高我想是不用擔心

森林綠拿來暈開好美，雖然是綠色又閃亮亮，但感覺是氣質奢華

香檳粉當然不用說，會是下眼線或打亮的好幫手

<a href="http://wenshing.pixnet.net/album/photo/189817266" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189817266</a>

這張照片下眼線就是使用香檳粉

偷偷說，我多買了一隻香檳粉，因為是NG商品才99元，當然馬上帶回家！

（只NG在外包裝不能吊在架上，對我來說根本不算NG！）



<span class="f6 hl">睫毛膏</span>
<a href="http://wenshing.pixnet.net/album/photo/189780788" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780788</a>
<span class="f3 hl">戀愛魔鏡  魅惑光感睫毛膏（二代）</span>

睫毛膏只能推都不打折的戀愛魔鏡了

這款是二代濃密，三代還沒有買，名稱很威的睫毛王本來要買後來也暫時滅火了

梳子型的刷頭我用得很順手

出門在外如果忘記帶睫毛梳，可以把沾取起來的睫毛膏擦掉後當睫毛梳使用

<a href="http://wenshing.pixnet.net/album/photo/189780792" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780792</a>

這隻目前已經快要用光了，所以刷起來就比較沒有那麼美

但我還是喜歡這隻啊！之後應該會去買三代濃密吧

畢竟從開始嘗試化妝到現在，永遠都會有一隻戀愛魔鏡在服役中

只是最近比較收斂，還不敢馬上購入三代

因為...未開封的睫毛膏中有kiss me的極太跟上一次的kitty貓眼睫毛膏啊！

等這兩隻用完或是濃密要出第四代，我一定會帶三代回家用用



另外，Fasio的絕世美睫也是我的愛，那款捲翹度跟刷起來的感覺都很棒！

用完後一直還沒補貨，怕之前買起來放的睫毛膏都沒有表現機會囉！

手邊用完後應該還是會回去買吧！我覺得有乖乖用眼唇卸妝液就不會難卸



<span class="f6 hl">眼部卸妝</span>
<a href="http://wenshing.pixnet.net/album/photo/189780776" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780776</a>
<span class="f3 hl">L'oreal  溫和眼唇卸妝液</span>

眼唇卸妝液當然要推L'oreal，中間有用過別款但還是最愛這款啊

尤其是有買一送一的時候最划算

目前剩下兩瓶，希望買一送一的活動快點回來！



<span class="f6 hl">臉部卸妝</span>
<a href="http://wenshing.pixnet.net/album/photo/189780756" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189780756</a>
<span class="f3 hl">CX卸妝水</span>

這款試看完佐拉文章後推薦給朋友使用，用完後反過來推薦我說好用的！

之前在康是美特價269就買了一瓶起來放，沒想到就聽到要停產了

（嚴格來說是換名字改包裝價格上漲，但似乎配方沒改）

所以趁出清特價一瓶199元的時候又買了五瓶！

目前抗暗沉型有4瓶，清爽跟保濕各1瓶，應該夠用一陣子了

另外，還跑去買了丸三化妝棉！濃一點的底妝要卸到沒顏色大概要用個三到四張吧

比起來絲花當然是舒服很多，可是價格也比較高

丸三則是經濟實惠啊！犧牲點舒適度也是OK的，而且還可以拿來濕敷真的很多用途

目前絲花則是拿來卸眼妝，眼妝我一定會使用絲花就是！



產品介紹完就來推一些小工具吧！

<a href="http://wenshing.pixnet.net/album/photo/189816928" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816928</a>

分別是，大創棉花棒、SOLONE腮紅刷、SOLONE睫毛梳、芮婷(?)燙睫毛器

<span class="f3 hl">大創棉花棒</span>

這真的挺不錯的，本身就是分開包裝，還有尖頭跟圓頭

放在化妝包帶出門要用很方便，不用另外再裝起來擔心髒掉

尖頭部分可以處理比較細小的地方，但出門我也不太常用到棉花棒，挺省的！

買一包花個39元就可以用好久



<span class="f3 hl">SOLONE的腮紅刷</span>

印象中是100出頭，這款我覺得毛挺舒服的

當初是為了刷NARS而買的，也買了有兩三年了吧！

<a href="http://wenshing.pixnet.net/album/photo/189816932" rel="nofollow" target="_blank">http://wenshing.pixnet.net/album/photo/189816932</a>
<span class="f3 hl">芮婷燙睫毛器（SASA購入）</span>

燙睫毛器對我來說是必備，因為我的睫毛很容易垂，就算是標榜捲翹的睫毛膏也一樣

有時候夾很翹，刷一刷有可能會慢慢的往下垂

如果真的不小心垂掉了，就要靠它來拯救一下啦！

朋友因為不太會使用睫毛夾，把這支給她嘗試過後就跑去SASA買一支了

不過她買的時候有特價，才199元！



<span class="f3 hl">SOLONE睫毛梳</span>

這睫毛鋼梳是我目前用起來順手的

之前買過大創的覺得不是很好用，因為腮紅刷對這牌有好感就順手買來用用看

沒想到意外的喜歡，只是之前去光南好像找不到了？

價格好像跟大創的差不多，也不貴就是！



這是我的2011年度彩妝，雖然用起來妝效不會是最好

但算是c/p值不錯且應該可以負擔的起

很多評價好的商品還沒有使用過，搞不好用完後我會變心也說不定

但那是2012年度彩妝的事情了！
