喜歡的請鼓勵，爸爸我也會因此而得到滿足。


內文與網誌相同，以下是網誌(無廣告)。
<a href="http://lazy-cellar.blogspot.tw/2014/01/diy.html" rel="nofollow" target="_blank">http://lazy-cellar.blogspot.tw/2014/01/diy.html</a>



那天孩子的娘在網路看到一個土製的盪鞦韆，是曬衣架水管塑膠椅做成的，
她一手指著照片，一手指著我的鼻子對著我說，你兒子也要一個，
沒想到要被人叫聲爹，是這麼的不容易，
於是身為爹，只好開始研究，最後參考了歪國人的做法，弄出了一個盪鞦韆，

看一看還真是高端、大氣、上檔次，實在有必要跟大家分享一下。

1. 先看成品照片後面比較好說明：
   <a href="http://ppt.cc/IWKP" rel="nofollow" target="_blank">http://ppt.cc/IWKP</a>

2. 骨架：
 　最重要的骨架，那四根木棍，我找了很久，甚至想過用窗簾桿，
 　因為國外的網站寫OAK，我去哪裡找橡木，
 　後來我在露Ｘ拍賣找到了紐松圓棒，但因我不了解紐松木的特性，
 　寶寶用的東西還是以安全考量優先，所以選擇了直徑4.3公分最粗的圓棒，
 　在此推薦《高Ｘ木業》，他們可以收費幫忙裁切及鑽孔，省了我好多麻煩，
 　(Ptt 上面就不貼連結了)
 　而且非常細心，電話簡訊聯絡我好幾次，服務真好，
 　兩根90公分圓棒各切一半，就是4根45公分的骨架，
 　並在每根骨架的頭尾4cm處各鑽一個1cm的孔。
 　送來就是下圖這樣，用細砂紙磨亮，把銳角磨圓。
   <a href="http://ppt.cc/uDpj" rel="nofollow" target="_blank">http://ppt.cc/uDpj</a>

3. 內側布料：
 　要買厚實一點的布，窗簾布、牛仔布之類的，比較安全，也比較好縫，
 　尺寸大約如下，建議先用紙剪下來比對，
 　黑色部分是實際的大小，紅色部分是留給包住木棍的長度，
   <a href="http://ppt.cc/4quc" rel="nofollow" target="_blank">http://ppt.cc/4quc</a>
 　實際做出來的樣子，這尺寸是依照我家寶寶的身型調整過的，大家可以再調整。
   <a href="http://ppt.cc/ssOw" rel="nofollow" target="_blank">http://ppt.cc/ssOw</a>

4. 繩子：
 　因為木棍鑽洞1公分，所以就買直徑1公分的繩子，可以上網選購特多龍繩，
 　3分=3/8吋=1cm，買10公尺應該很夠了，我買了100公尺，哈哈，
 　剩下拿來綁樓梯扶手的縫隙，我覺得1公分的繩子不論安全與美觀都是最剛好的。
   <a href="http://ppt.cc/w4Op" rel="nofollow" target="_blank">http://ppt.cc/w4Op</a>

5. 支點：
 　找個牢靠的地方，綁平結，我綁在陽台鐵欄杆上面，對面鄰居跑出來關心，
   <a href="http://ppt.cc/QABF" rel="nofollow" target="_blank">http://ppt.cc/QABF</a>

   繩與繩接點是白鐵製8mm的快接環，很粗勇，不要買像掛鑰匙圈那種鋁合金，
　 買白鐵的就對了，如果要買鋁合金的，就買登山專用的，
   <a href="http://ppt.cc/zQ9h" rel="nofollow" target="_blank">http://ppt.cc/zQ9h</a>
　
 　接下來就是組合起來，木頭穿過布，線穿過洞，然後把線打個結，
 　我有加一個小墊片保護木頭，這樣就完成了。
   <a href="http://ppt.cc/YYQn" rel="nofollow" target="_blank">http://ppt.cc/YYQn</a>

6. 小地方：
 　把扣環下面綁起來，可以固定盪鞦韆的水平角度，繩子比較不會亂跑。
   <a href="http://ppt.cc/1GrM" rel="nofollow" target="_blank">http://ppt.cc/1GrM</a>
