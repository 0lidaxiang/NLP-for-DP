


由於阿苗4M就長了兩顆牙，兩媽很早就見識到長牙歡的嬰兒有多恐怖

在親友的建議下給了一些冰涼的水果泥，果然有改善他歡歡的程度

於是也讓媽咪踏上了做果凍、布丁的不歸路...

在這邊也分享一下食譜給需要做點心的家長們～








<span class="f6 hl">＊南瓜/紫蕃薯雙色布丁</span> (50ml/四份)
  圖片：<a href="http://ppt.cc/pxM8" rel="nofollow" target="_blank">http://ppt.cc/pxM8</a> <a href="http://ppt.cc/DNMD" rel="nofollow" target="_blank">http://ppt.cc/DNMD</a>

  原料：紫色地瓜泥50ml/南瓜泥150ml/吉利丁2片/50ml的模具X4
  步驟：取兩個小碗分別放入南瓜/地瓜泥，在南瓜泥內放入剪碎的一片半吉利丁
        在地瓜泥內放入半片吉利丁，將兩碗一起隔水加熱到吉利丁片融化後攪拌均勻

        先在容器內鋪好比較多的南瓜泥，然後連模具一起放入冷凍庫冰10~15分鐘
        接著再鋪上剩下的地瓜泥放入冷藏凝固。脫膜時將模具稍微泡熱水會比較輕鬆。
       （建議冰久一點，不然脫膜時可能會像我這樣不夠漂亮）

<span class="hl">  &lt;由於阿苗對於雙色系列很無感，後來嫌麻煩就沒做過要脫膜的多色系列布丁</span>&gt;



<span class="f6 hl">＊蘋果果凍</span>(100ml/二份) 與 <span class="f6 hl">南瓜牛奶布丁</span>(100ml/二份)
  圖片：<a href="http://ppt.cc/nVOY" rel="nofollow" target="_blank">http://ppt.cc/nVOY</a>

  原料：鮮榨蘋果汁150ml/南瓜泥100ml/已沖好配方奶100ml/吉利丁兩片
  蘋果果凍步驟：一片吉利丁剪碎，與50ml的水放在小碗內隔水加熱融化完全。
                再把吉利丁水與蘋果汁混合裝瓶封蓋冷藏，等待凝固即可食用。
  南瓜牛奶布丁步驟：南瓜泥與一片剪碎的吉利丁一起隔水加熱，等待吉利丁融化完全
                    將把南瓜泥與配方奶混合均勻後裝瓶封蓋冷藏，凝固後即可食用。

<span class="hl">  &lt;蘋果汁是阿苗不肯喝的東西，做成果凍之後他就變得好愛吃&gt;
</span><span class="hl">  &lt;南瓜布丁也可以改用雞蛋作為凝固劑&gt;
</span>
<span class="f6 hl">＊香蕉牛奶布丁</span>(100ml/四份)
  圖片：<a href="http://ppt.cc/h6oj" rel="nofollow" target="_blank">http://ppt.cc/h6oj</a>

  原料：香蕉泥200ml/已經衝泡好的配方奶200ml/吉利丁2片
  步驟：吉利丁剪碎與配方奶隔水加熱融化，再與香蕉泥一起攪拌均勻放入冷藏
        等待凝固後即可食用。

<span class="hl">  &lt;香蕉不小心買太多，偶而有一兩根過熟時可以做的點心&gt;
</span>
<span class="f6 hl">＊芝麻奶酪</span>（100ml/3份）
  圖片：<a href="http://ppt.cc/8Ies" rel="nofollow" target="_blank">http://ppt.cc/8Ies</a>

  原料：黑芝麻兩三匙/已沖好配方奶250ml/吉利丁一片半
  步驟：黑芝麻放入小鍋子內稍微炒過，再加入吉利丁與配方奶繼續加熱攪拌均勻
        直到吉利丁與奶粉完全溶解後，再用攪拌棒打過讓芝麻顆粒更細碎
        最後裝瓶封蓋冷藏，凝固後就可以讓寶寶吃了

<span class="hl">  &lt;黑芝麻對於補充鈣質有很大的幫助，所以刻意做芝麻口味希望苗苗快長出牙齒不要歡&gt;
</span>
<span class="f6 hl">＊葡萄果凍</span>（100ml/3份）
  圖片：<a href="http://ppt.cc/ddUm" rel="nofollow" target="_blank">http://ppt.cc/ddUm</a> <a href="http://ppt.cc/R6x9" rel="nofollow" target="_blank">http://ppt.cc/R6x9</a>

  原料：含果肉的葡萄汁150～200ml/吉利丁一片半
  步驟：手工壓榨的新鮮無子葡萄汁先準備好，與水調成適當的甜度
        將吉利丁剪碎放置小碗內，加少許水隔水加熱融化完成
        混合葡萄汁與吉利丁液之後裝瓶封蓋冷藏，等待凝固之後即可食用。

<span class="hl">&lt;由於葡萄汁實在太甜了，所以媽咪還是先調整成自己覺得OK的程度才給寶寶吃&gt;</span>





Q：吉利丁是什麼？ 哪裡買？
A：吉利丁是動物膠質，從魚的鱗片、動物軟骨中提煉取得，跟女性常喝的膠原蛋白差不多
   一般在烘焙材料坊都可以買到，建議購買國外比較知名品牌的吉利丁比較安全

Q：如果不要用吉利丁可以做出果凍、布丁嗎？
A：果凍可以用洋菜粉、珊瑚草代替，但是兩媽居住的廣東沒有賣，所以無法提供配方比例
   布丁部分如果是做成熱食甜點，則可以參考法式布蕾的方式用雞蛋代替。
