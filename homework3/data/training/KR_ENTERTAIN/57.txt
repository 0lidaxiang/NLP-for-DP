這次旅行的開始 是製作組送任務卡到每個人的家裡 <span class="f0 hl"> (突然有種RM的感覺XD</span>

˙民國家
<span class="f2">民律弟弟:「每天每天每天每天每天都在睡帳篷」</span>
 一句話說明了自己哥哥和爸爸聽完任務的無奈XDDD
打電話討救兵的爸爸找上滑冰選手雖然順利借到了自以為發橫財了 殊不知......

˙成俊家
可惜成爸爸是在拍戲現場接到任務 想看俊兒的反應阿
<span class="f3">車太賢:「這我們基本模式阿」XDD</span> 你也帶著小孩來參加吧XDD
還要求馬賽克 這邊超有趣的~
成東日感覺就是很有經驗的老手關於冬季露營

˙尹厚家
細心的爸爸寫下需求 但<span class="f6">厚滿心滿腦都是吃的</span>( ′-`)y-～
厚運用自己廣大的人脈 借到了帳篷

˙智雅家
可愛的智旭弟弟不斷的探頭 宋爸爸認識的人都在國外 怎麼辦呢XD

˙俊秀家
一開門就是四次元的<span class="f1">是是是是是是~連任務紙都撕開了</span>(？o？)
但天真無邪的俊秀絲毫不關心阿 哈哈哈 還要用雪做帳篷
長得也很俊俏的李家哥哥鼓吹著爸爸 不如直接買一套吧~
<span class="f1">看似很散漫卻意外用功的爸爸臨行前還實地練習搭帳篷呢</span>

===============到了出發當日==================

˙俊秀家
剛睡醒發懵的俊秀實在是太可愛了wwwwwwwwwww
<span class="f1 hl">與爸爸的腿產生愛情更是一絕XD</span> 這裡真是太可愛了!!還能當人體拖把
還好他家地板夠乾淨XDDD還被拖去洗臉
<span class="f1 hl">用耳帽擦自己的太陽眼鏡的阿爸也很好笑XDDD</span>

˙成俊家
東日阿爸自豪的暖手袋工廠(~^O^~)   <span class="f3 hl">俊兒用崇拜的眼神看著阿爸 超有愛的wwwwwww</span>
連阿爸的手套都貼心的放在包包裡&lt;3&lt;3
<span class="f3 hl">最後融化姨母的防線!!是俊兒興奮的發語詞!!!</span>
自己開心的推著行李貼近螢幕 都超可愛的!!!!

˙尹厚家
還在昏迷的厚被爸爸強行拖離床鋪卻又在沙發昏迷了XDDD
最後是靠著酸奶和阿爸的搔癢醒過來 <span class="f6 hl">在車上一直一直玩得很開心的歌曲接唱</span>
<span class="f6 hl">真的很喜歡父子倆歌唱時間</span>

˙智雅家
乾脆買了帳篷的鐘國阿爸 帶了好多好多給司XDD
智旭弟弟突然醒來  但只能帶著對兒子的抱歉出發了!

˙民國家
意外的是男主人開門 <span class="f2">媽媽生了小寶寶(粗咖黑!)</span>去了坐月子中心
<span class="f2 hl">可憐的民國 被爸爸洗頭 一會冷一會熱(－^－)</span> 換衣服的時候也很"落漆"
出發前還炫耀了快搭帳篷......自信滿滿的!
最後連換鞋子都......<span class="f2 hl">民國一定很想念媽媽阿~~~~~</span> 出發前就有點坎坷
說著不去零下14度的春川最好了 沒想到車卻駛向了春川湖阿~~~~~~
<span class="f2 hl">好像在預告成柱阿爸未來的命運.......</span>


===============到達露營地點==============

<span class="hl">搭帳篷對決</span>

快搭帳篷組很快就搭好了讓成柱阿爸很得意 到處「巡查」
<span class="f6">得意不到五分鐘就發現事情大條了!!!!</span> <span class="f0 hl">我在想金東聖選手也沒有制止成柱阿爸嗎</span>
就算是最後一名也高興的俊兒更享受和爸爸兩人時光 <span class="f5 hl">撒嬌的俊兒超可愛的/////// </span>

東日阿爸還壞心的將快搭扔了 民國帶著報復心態還去搖了搖東日家
<span class="f6">但是快搭一扔就很遠了 東日家怎麼搖都不會壞阿!!!</span>
<span class="f5 hl">貼心的智雅還去安慰他們</span> 可惜於事無補......
<span class="f6 hl">民國還繞得遠遠去牆角劃圈圈</span>ˋ(′_‵||)ˊ <span class="f0 hl">對不起我笑了￣▽￣||</span>
<span class="f5 hl">尹厚真的超好笑的 一直賴在智雅家卻不去幫自己的爸爸XDD</span>
<span class="f5">標準的吃裡扒外代表</span> 還藉機跟智雅發展LOVELINE

看過別家的帳篷連成柱爸爸都要哭了......<span class="f0 hl">身為觀眾的我真的覺得又好氣又好笑又同情</span>
<span class="f6 hl">民國為了忍住眼淚還要狂奔向遠方!!! 後製上字也太好笑了「老天阿~」</span>
<span class="f6 hl">努力不哭 說著「天氣真好」的民國阿~</span> 連我都忍不住想笑了 再次對不起阿

使喚著俊兒的爸爸 硬是要與俊兒上演著愛情戲的阿爸
我內心大喊著<span class="f1 hl">我也要阿!!!!</span> <span class="f2">我也想要和俊兒踩腳印 也想要大喊撒朗黑 也想要在雪地翻滾</span>
看戲的鐘赫父子也好好笑 <span class="f3">俊秀眼神透露出 「我也想要試試看」的激光XD</span>

終於民國還是忍不住眼淚在快搭裡爆炸了QQQQQQQQ <span class="f0 hl">我猜他可能覺得在帳篷裡哭沒關係</span>
<span class="f3">「知道為什麼哭嗎因為房子太小了」智雅還跑來補上一刀</span> <span class="f0 hl">我又很沒良心的笑翻了~~~</span>

在我看來  <span class="f2">還是成柱阿爸的失誤比較大阿Orzzzzzzzzz</span>但成柱阿爸也有不得以的苦衷...
要工作又要照顧媽媽還有三個小孩(2+1) 真的很有可能手忙腳亂
<span class="f1 hl">大家一起幫民國家搭建帳篷的時候真的好溫馨喔TTTTTTTTTTT</span>
阿爸們雖然嘴巴有點壞 還一直虧他們 但還是搭了個溫暖的家
<span class="f1 hl">很喜歡這個部份!!</span>大家互相照顧 解決問題~
