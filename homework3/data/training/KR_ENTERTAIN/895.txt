李基燦爆料趙正治大學時代的超旺人氣。

24日下午播出的Mnet「The Beatles Code 2」中，大學同學趙正治和李基燦一同出演了
節目，互相爆料對方的大學時代，吸引了眾人的視線。

當天的節目中，趙正治被問理想型時回答說「沒有特別的理想型。Jung In不是我的理
想型。只要漂亮就可以了。最近的話總是會注意到腿好看的異性」。

接著，李基燦爆料說「趙正治大學的時候身邊的女人從來沒斷過。上學的時候他的女友
和現在的Jung In是完全相反的類型」，引發了現場爆笑。

而趙正治果然不甘其後「李基燦大學的時候也很厲害呢。我知道的可不少，所以我們還
是不要互揭老底了」，對李基燦進行了「警告」。

另外，在當天的演唱夾雜著回憶和故事的歌曲環節「音樂學概論」中，李基燦和白娥娟
合唱了Luther Vandross和瑪麗亞凱莉的「Endless Love」，而Kim Yae Lim在趙正治的
吉他伴奏下演唱了瑪麗蓮夢露的「I Wanna Be Loved by You」

24日播出的Mnet「The Beatles Code 2」中，歌手白娥娟出演，開口道「對外貌很不滿
足」，「也有想去做整形手術的想法」，吸引眾人關注。

當天的節目中，白娥娟就自己的外貌表示「特別是因為單眼皮的關係，對雙眼皮手術格
外關心」，對此，主持人神童就自己的經驗飽含真心地回應道「雙眼皮手術很好」，「
我能推薦給你不錯的地方」，笑翻全場。

不過，白娥娟就整容的疑惑劃清了界線。她坦率表示「我目前還沒做過任何整形手術」
，「只做過牙齒矯正」。

另外，白娥娟最近發表了新歌，正展開積極的活動中。

24日播出的Mnet「The Beatles Code 2」中，歌手李基燦、趙正治、Kim Yae Lim、白
娥娟出演。

當天，Kim Yae Lim賣力熱唱了最近發表的新歌「All Right」。聽完後的主持人張東民
說道「現在是嘴裏含著糖唱歌嗎？好神奇，嘴巴一邊慢慢地蠕動一邊唱歌」，吸引眾人
關注。

另外，身旁的另一位主持人則表示「想起了演員朱鉉老師」，說完用朱鉉的聲帶模仿跟
唱了Kim Yae Lim的歌曲，笑翻全場。

對此，網友們紛紛表示「張東民對Kim Yae Lim的毒舌，張東民的反應太好笑了」，「
張東民對Kim Yae Lim的毒舌，竟被說成是嘴裏含糖」，「張東民對Kim Yae Lim的毒舌
也好有才」等。

此外，當天的節目中，Kim Yae Lim還選出演員孔劉為自己的理想型。
