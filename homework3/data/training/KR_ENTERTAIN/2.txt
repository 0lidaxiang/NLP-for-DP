
身為RM忠實觀眾 沒想到第一次發文竟是因為這個節目XD

因為等待720P中字第五集(感謝LPJames大) 這個空檔

所以就想說那來發個心得文吧~



簡言之 就是個爸爸帶著小孩去玩的節目 <span class="f3">在沒有媽媽的情況下!</span>後面這句很重要 XD

孩子大部分都是黏媽媽的  再加上演藝人員這份職業

藝人們的孩子可能認得「電視裡的爸爸」卻無法親近現實生活中的爸爸

看了四集下來 真的可以理解為什麼這個節目會竄紅了!!

<span class="f5 hl">孩子們可愛絕對是關鍵</span> <span class="f3 hl">每個爸爸和小朋友的相處也是一大重點</span>

它不像RM同樣戶外型綜藝 一個接著一個的緊湊遊戲

也不是棚內綜藝 大家聚在一起討論一個話題


目前1~4集看下來 主要共同任務也就只有

<span class="f2">第一天選房子→孩子要食材爸爸做晚餐→睡覺→</span>
<span class="f2">第二天爸爸早起作早餐→活動(EP2雪橇EP4隱藏攝影機)</span>

感覺好像沒做什麼事，但其實小孩和爸爸卻在這之中拉近不少距離

可能顧慮到孩子體力和爸爸的 安排的活動也沒有很多

所以很多都是在自己家相處情形 但就是因為每家有每家獨特的風格

才讓人對這節目著迷了:) 下面就淺談每對父子的個人感想XD


1.父:李鐘赫(演員；去年作品紳品花美中年F4之一)
  子:李俊秀(B型；血型似乎跟人很符合)

  這對父子真的很有趣！感覺李鐘赫對俊秀的教育比較自由開放，有點樂天派

  所以才能發展四次元的俊秀，外加有點盧XDDDDD 一直問著為什麼 一直叫阿爸

  問到鐘赫阿爸都煩了(笑) <span class="f6 hl">可是鐘赫阿爸也沒怎麼生氣 這是我很欣賞的地方</span>
<span class="f6 hl">俊秀笑起來的時候瞇瞇眼真的好可愛阿~~~~~~~~~~~~~</span>

  還有這小子在承認自己說謊時 <span class="f6 hl">還微笑+電眼放送XDD</span> 鐘赫阿爸不原諒都難吧!!!


2.父:尹民秀(歌手；歌手生的兒子好像也很會唱)
  子:尹厚  (吃飯一定吃得最香的那個)

  <span class="f4 hl">尹厚最會吃的小朋友來著XDD</span>看那嬰兒肥的臉頰 可能很多姨母飯都想捏一下!

  這對父子就跟媽媽說的 沒有媽媽也沒關係 <span class="f4 hl">一搭一唱很合拍~</span>

  雖然在第三集雞蛋那邊讓我在心理扣了一點分

  如果老實承認自己抗拒不了誘惑 所以不小心貪吃了不是更可愛嗎\(@^0^@)/

  還蠻喜歡父子一起唱歌的時候 真的誰生的還是會遺傳

  還有可愛又神奇的LOVELINE 基於我是哥哥要照顧妹妹的想法

  其實很想跟厚說:<span class="f4 hl">「你不是哥哥，你是選手吧(指)!!!!!!」</span>


3.父:金成朱(播音員；大家在一起時就會擔任主持人角色)
  子:金民國(孩子們中最大的哥哥；也最愛哭)

  運氣最不好的父子 前面的抽籤 後面比賽動作慢 <span class="f5 hl">擔當悲情角色的成朱民國父子</span>
<span class="f5 hl">其實我還蠻喜歡他們的</span> 可能有些人覺得民國很愛哭有點煩

  但我覺得哭只是民國宣洩情緒的方式 <span class="f5 hl">在不哭的時候可是個超有禮貌很講義氣的孩子阿</span>

  我自己在教育現場遇過很盧的孩子 民國比起來真的一點都不算什麼了

  民國哭完就接受了 而且哭也不是什麼壞事阿 就像成朱阿爸因為害怕而挽留兒子在身邊

  我也覺得這不是什麼壞事阿 偶爾跟孩子示弱 可以讓自己跟孩子更親近

  <span class="f5 hl">讓孩子覺得自己也是可以被別人依靠的</span> 孩子會懂得照顧人 也會有同理心 這不是很好嗎

  每個人都會有想哭的時候 每個人也都會有害怕的時候

  很期待這對父子後面還會有什麼發展


4.父:宋鐘國(足球選手；好像是當年世界盃有印象......)
  女:宋智雅(萬綠叢中一點紅；跟厚哥哥有曖昧的可能)

  宋鐘國完全就是女兒控阿!!!!!!!<span class="f2 hl">「女兒真是上輩子的情人」</span>

  這句話最適合鐘國阿爸了XDDDD生活起居真的照顧的無微不至 基本的背背抱抱

  擦臉洗腳一定要的 <span class="f2 hl">就連外面的男人都要牽制一下</span>\⊙▽⊙/

  智雅其實也很會應付阿爸XDD 每次看到鐘國阿爸在邀功「這我煮的飯喔」

  絕對不會吝嗇她的讚 <span class="f2 hl">也很會甜言蜜語 完全就是把阿爸吃得死死嘛XDDDD</span>

  我本來以為不吃飯的時候鐘國阿爸會狠心 但智雅一哭還是不行阿( ￣ c￣)y▂ξ

  他們家的黑臉一定是媽媽XDDD


5.父:成東日(演員；去年作品回應吧1997詩源的爸爸感覺跟戲裡有點像)
  子:成俊  (人如其名的正太，從cool kid變成sweet kid)

  雙俊擄獲不少姨母飯，<span class="f1 hl">成俊魅力尤其大!!!</span>  臉蛋真的很優

  尤其是<span class="f1 hl">從原本酷酷的臉變成笑起來很帥的模樣 我喜歡反差阿XDDDD</span>

  感覺參加這個節目收穫最多的是這對父子 <span class="f1 hl">從原本的不熟到現在會說撒朗黑!</span>

  成東日就是個刀子嘴豆腐心的人又有點小奸小惡 但很真實

  可能有人擔心孩子的教育怎辦 但其實我們現實的社會就是如此 <span class="f1 hl">有誰能不自私呢</span>

  成東日老來得子(?! 但也沒有很寵成俊 反而是嚴格對待 也造成父子間的疏離

  還好透過旅行透過活動 俊兒一定可以感受到東日阿爸滿滿的愛

  不知道之後俊兒跟東日阿爸還會有什麼變化 期待!!



聽說韓國好像做到十集了(?) <span class="f3">疲倦的時候看看小朋友 真的很有治癒的功能</span>
<span class="f5">看看智雅的撒嬌</span> <span class="f4">看看厚吃得多麼香</span> <span class="f1">看看俊秀的笑臉</span>
<span class="f6">看看民國多有禮貌</span> <span class="f2">看看俊兒帥氣的臉蛋</span>

還有各個阿爸和小朋友間的相處 看看他們的有趣事情 看著看著真的越看越入迷呢


現在上韓綜版都好期待下集的中字趕快出來XDDDDDDD

大家一起來討論這五個可愛的孩子和有趣的阿爸們吧(≧&lt;&gt;≦)
