話說我第一次看到車仁表大叔不是在戲中，而是02年的MBC演藝大賞，
他是前一年的大賞得主，頒獎給當年的大賞得主張瑞希<span class="f0 hl">﹝人魚小姐﹞</span>，
當時真的覺得他好帥啊！恩，就醬 XD。
<span class="f0 hl">﹝畢竟我是因為張瑞希才看那是視頻的﹞</span>

第一次看車大叔演的戲則是因為智孝，2011年的《階伯》中演男主角的父親，
<span class="f0 hl">﹝但車大叔跟智孝是在不同的時代所以沒對過戲，可惜可惜﹞</span>
那是我第一次見識到車大叔的超讚演技跟傲人體魄，果然是元祖肌肉男！
不過當時就有聽說，<span class="f3 hl">車大叔外表粗爌，內心卻很大孩子 XD</span>，
恩，這集還真是見識到了，傳聞其來有自啊！

這集車大叔完全是主持病發作，加上超級大又多的動作，
畫片那邊真的太好笑了，Pose一大堆結果…打空！XD
很少看到有比在石還要話澇的，而且這集他還不是唯一一個…

徐章勳的嘮叨就這樣持續了整集，搭配這集特別調皮不斷調戲他的哈哈，
<span class="f3 hl">明明非常不會玩卻超級有存在感</span>，不得不說哈哈的功力又上了一層，
搶椅子那邊真的看得出來他很紳士，畢竟以他的體能要把所有人都推開很容易，
<span class="f0 hl">﹝忽然想起當年太陽跟火箭打群架，Shaq輕輕鬆鬆把所有人推出去的畫面 XD﹞</span>
如果讓智孝他們那組先過關，單純面對鍾國 + Ricky、在石 + 車仁表，
徐章勳應該會比較放得開，<span class="f3 hl">我想他真的很怕把女孩子弄傷…</span><span class="f0 hl">﹝畢竟智孝太拼了 XD﹞</span>

Ricky這集就很可惜了，感覺他就是來跟老友敘舊的，畫面非常少，
占了很多篇幅的火車關卡，其他四組實在太搶鏡導致這組幾乎都被通編，
雖然前面有跟能力者對決的畫面，但後面畫片王之戰又是幾乎沒出賽…
<span class="f3 hl">副族長你應該出演下一集的才對啊！再把柳譚或光熙找來湊足七人剛剛好！</span>
<span class="f0 hl">                                        ﹝副族長：我是先來探路的﹞</span>

最後的畫片王之戰雖然說已經不是第一次，可能很多人會覺得有點膩了，
但我還是覺得很刺激，特別是看兩大山脈對決了一個小時甚至到換畫片的程度，
<span class="f3 hl">隔著螢幕都能感覺到現場他們那種燃燒勝負欲的熱度，</span>
在石這集真的不愧畫片王稱號，除了對鍾國那場<span class="f6 hl">勝負神在開玩笑</span>兩人都翻不過來外，
其他場次幾乎都是快速解決，攻無不克的強大啊！

畫片最弱者之戰也很好笑，<span class="f5 hl">智孝的畫片實力還真是從來就沒進步過啊…。</span>
<span class="f0 hl">﹝這場有可能會比畫片王之戰更久 XD，技術太差更翻不過來﹞</span>


本來兩週前看到來賓時很期待，今天看到收視率又很失落擔心不好看，
看完之後…這集其實真的不錯，<span class="f3 hl">但RM的收視好像真的是最會被假日影響的</span>，
<span class="f0 hl">﹝可能跟收視族群有關，之前看過統計RM在30歲以下的收看率是超強的﹞</span>
我記得農曆過年那集<span class="f0 hl">﹝EP132﹞</span>一掉就是6%，去年中秋EP113超好笑也掉很慘，
<span class="f3 hl">只是這周忽然出現一個自兩年前EP50泰國特輯後就沒出現過的最低12.7%，</span>
讓人非常非常非常傻眼就是……下集如果還是這麼慘再擔心吧！
<span class="f0 hl">﹝另外就是一部《赤腳的朋友們》也未免太慘了，2.9%！
</span><span class="f0 hl">　家族二都沒有這麼慘過，這個節目到底發生了甚麼事情……﹞</span>

話說<span class="f3 hl">前段追擊戰、後段畫片王之爭似乎已經成為體能強者出演時的固定關卡了</span>，
EP119秋信守柳賢鎮、EP131秋成勳李詩英，都是幾乎相同的節目結構，
開頭追擊戰 -&gt; 一些遊戲 -&gt; 畫片，只是EP119還多了超能力棒球，
其實這樣還不錯，<span class="f3 hl">因為畫片是最能平衡體能上大落差又能保持刺激度的</span>；
畫片的重複率其實不是問題，因為事實是，畫片最終戰還是超刺激，
比起畫片，其實最近最常出現的遊戲是躲避球…XDD。<span class="f0 hl">﹝EP139、143、144﹞</span>

這集如果說有甚麼缺點，那就是<span class="f3 hl">火車關卡實在是太不刺激了</span>，
以往即使是輕鬆旅遊間進行遊戲，也還是有一定程度的勝負感，
這集的火車關卡感覺上一點勝負欲都沒有，就是家人出遊 XD，
<span class="f3 hl">而且沒有超車的可能性，全部的關卡都是One Chance也影響了精采度</span>，
雖然說看調皮哈哈跟拘謹徐章勳吵嘴、還有整個活潑到爆炸的車大叔很好笑，
還有閃到爆炸的周一情侶真的就是出來郊遊的，但遊戲精采度也得有起碼的低標啊！

還有感覺最近RM越來越不在觀眾前進行遊戲了，這集只踢了鞋子，
之前幾集的躲避球、水上關卡、這集的火車跟搶椅子都是清場狀態，
但最大的問題是，<span class="f3 hl">沒有新意的遊戲或玩的不好的遊戲比例增加了</span>，
像上一集的阿凡達追擊戰就屬於沒有玩得有些失敗的遊戲，
這集火車小遊戲則沒甚麼爆點，<span class="f3 hl">互動似乎比遊戲本身更有看頭</span>，
EP141的水砲彈猜謎也是互動比遊戲有看頭，製作組要加油啊！

說難聽點，上次海外大特輯似乎耗掉製作組太多心力，最近都在放電 XD，
<span class="f1 hl">EP139之後的六集在我看來只有Couple特輯的遊戲有「靈魂」</span>，
連RM最大賣點的無來賓個人戰EP140《The Virus》都有點不盡人意，
加上上集李敬奎的置入行銷太過、這集又正好撞上放假，
MBC的綜藝復活也瓜分了收視群，連鎖效應造成兩年來最低收視率…
看看下集《叢林的法則》上門踢館<span class="f0 hl">﹝幫忙﹞</span>能不能把收視率拉回水平~~~~


話說最近智孝雖然拍戲綜藝兩頭燒，黑眼圈越來越重<span class="f0 hl">﹝還好氣色還可以﹞</span>，
但卻不像去年或前年那樣幾乎沒份量，最近幾集反而超有存在感，
即使EP137有Uie、EP139有$M兩大美女、EP141有Jessica、EP142有李寶英，
智孝卻還是靠著強大遊戲實力跟家人<span class="f0 hl">﹝情侶﹞</span>互動吸了大量目光，
難怪李敬奎可以用「救活了80%的Running Man」這種話來稱讚她，
當然沒有這麼誇張，但智孝真的是女鐵人來的~~~~。<span class="f0 hl">﹝果然是曾嘗試過三鐵的女人﹞</span>
<span class="f5 hl">下集叢林女戰士全慧彬上門踢館，智孝千萬別輸了啊！</span>

話說以往像奧運特輯<span class="f0 hl">﹝EP104、EP129﹞</span>和紳品高中<span class="f0 hl">﹝EP138﹞</span>這種團隊踢館戰，
男子選手的體能都是來賓佔上風，但RM會在女子選手部分佔到優勢<span class="f0 hl">﹝ACE智孝﹞</span>，
