風靡全球的小說暨電影《暮光之城》作者史蒂芬妮梅爾在《暮光之城》又推出了融合浪漫

愛情與外星科幻元素的全新鉅作《宿主》，改編成電影後電影公司找來了Saoirse Ronan

莎柔斯蘿南 演出劇中的梅蘭妮，等等，莎柔斯蘿南？你一定有所懷疑，因為也許你根本

就不記得她！那麼快點來了解更多有關她的大小事吧！

<a href="http://tinyurl.com/c2pqajh" rel="nofollow" target="_blank">http://tinyurl.com/c2pqajh</a>

愛爾蘭女演員Saoirse Ronan 莎柔斯蘿南 是好萊塢即將崛起的新星，但是她卻說如果她

朋友開始以好萊塢大明星來對待她、和她保持距離，她會很不開心。

<a href="http://tinyurl.com/bsg9sa9" rel="nofollow" target="_blank">http://tinyurl.com/bsg9sa9</a>
<a href="http://tinyurl.com/cze5cb7" rel="nofollow" target="_blank">http://tinyurl.com/cze5cb7</a>

想起來了嗎？對啦，她就是演《蘇西的世界》裡的蘇西。

<a href="http://tinyurl.com/blsxm4p" rel="nofollow" target="_blank">http://tinyurl.com/blsxm4p</a>

年僅19歲的她除了《蘇西的世界》外，也有不少的作品，例如：《少女殺手的奇幻旅程》

以及《贖罪》，甚至還獲得奧斯卡提名過。雖然還是花漾少女的年紀，卻已經有十分超齡

的演出

<a href="http://tinyurl.com/bmnac9o" rel="nofollow" target="_blank">http://tinyurl.com/bmnac9o</a>

在新電影《宿主》中，莎柔斯蘿南演出身體住了兩個靈魂的少女，因此她在戲中必須和自

己對話，差點沒把她搞到精神分裂；還有一幕，需要她從高達6樓的地方跳下，大家都覺

得她一定會使用替身，沒想到莎柔斯蘿南堅持要親自上陣，跌破大家的眼鏡。

<a href="http://tinyurl.com/brs2j6z" rel="nofollow" target="_blank">http://tinyurl.com/brs2j6z</a>

一直想在電影裡單純談個戀愛的她終於在新片《宿主》圓夢，而且還接替克莉絲汀史都華

演出《暮光之城》作者史蒂芬妮梅爾的浪漫小說，雖然她的外表與克莉絲汀史都華大不同

，但她會不會成為克莉絲汀史都華的接班人呢？

<a href="http://tinyurl.com/cajwbfc" rel="nofollow" target="_blank">http://tinyurl.com/cajwbfc</a>

當然媒體也不得不將她所演的《宿主》和《暮光之城》拿來比較一番。被問到「你會不會

擔心妳演的《宿主》不如《暮光之城》呢？」她說：「真的沒有想過耶！我知道《暮光之

城》讓全球掀起了一陣風潮，不過《宿主》比較不像那種類型的電影，所以沒有什麼好比

較，也沒有什麼好擔心的。」

<a href="http://tinyurl.com/c6agrnh" rel="nofollow" target="_blank">http://tinyurl.com/c6agrnh</a>

電影《宿主》講述著由莎柔斯蘿南所飾演的女主角因為拒絕靈魂被侵占，因此與外星物種

發生衝突，但是如果她有一天能夠入侵別人的靈魂，寄宿在別人的身體中，她會想要住在

誰的身體裡呢？她答道：「大概是碧昂絲吧！因為她身材超好的！根本是女神吧！」

<a href="http://tinyurl.com/d3ov8yv" rel="nofollow" target="_blank">http://tinyurl.com/d3ov8yv</a>

那麼莎柔斯蘿南演完《宿主》後，會有什麼計畫嗎？

「我最近會準備和萊恩葛斯林一起開拍新片《How to Catch a Monster》，我超興奮的！

」(不知道是很興奮要拍新片還是很興奮可以跟帥哥…..？)
