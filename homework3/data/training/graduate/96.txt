我有補習，6科全補

因為其實大一二上課的時候都只是顧60而已

對於要考的科目幾乎都沒甚麼記憶

所以補習對我來說是比較適合的一條路

但後來自己讀一讀念一念之後反而覺得有些科目不用補沒關係

在後面會詳敘


以下是各科準備方式還有一些小心得

可能不適用於每一個人，參考就好



--

<span class="f3 hl">離散數學:</span>

重點章節大家都清楚
不外乎排列組合、遞迴、生成函數、圖、二元關係等等
題目一定要很熟 看到類似題型馬上就知道要怎麼解
似懂非懂的話進考場一定會慌
像今年台大1~50挑7個數字不相鄰那題我就寫錯了XD

假如是用林緯講義在準備的同學
建議去買一下黃子嘉的書
林緯的書後面題目有些都還蠻舊的
感覺比較沒有參考性

另外台大的群環體證明其實都還蠻簡單的
有空的話一定要準備 寫出來你就贏過大半考生了!

書籍:林緯講義四本

觀念重要性:<span class="f1 hl">高</span> 題目重要性:<span class="f1 hl">高</span>

--

<span class="f3 hl">線性代數:</span>

台南大碩教線代的老師嘛..恩
我上了兩堂就決定自己看了

小黃的講義 1,2,3,4,5,7章一定要看
是說我也只有看這幾章 第八章準備太花時間
線代這東西看第一遍的時候通常會不知道在講什麼
但第二遍第三遍開始融會貫通速度就快了!
每多看一次都會有一種:「啊，原來如此」的感覺

資工所考得比較沒那麼深
計算弄熟，知道什麼時候用甚麼方法就ok
不用太拘泥於證明題，觀念弄懂才是真的
交大的線代都蠻有水準的，可以用來複習!

書籍:黃子嘉講義(上)(下)

觀念重要性:<span class="f1 hl">高</span> 題目重要性:<span class="f2 hl">中</span>

--

<span class="f3 hl">計算機組織:</span>

高銘計組上得還不錯
但跳來跳去還蠻討厭的
而且進度偏慢，建議是自己念
上課時再釐清自己觀念跟問問題

重點依然大家都知道
datapath、virtual memory等等
所以也不多說了 題目一定要做
之前準備的時候都只是看過觀念
想說題目都一個樣子
但常常寫完某個題目後
就發現自己的觀念完全錯誤!
為了不多走冤枉路，第一次看時就該配合題目釐清觀念!

書籍:高銘講義(上)(下)

觀念重要性:<span class="f2 hl">中</span> 題目重要性:<span class="f1 hl">高</span>

<span class="f3 hl">OS:</span>

很玄的科目
考難大家都不會、考簡單每個人都滿分
重點一樣都那幾個

高銘的OS就是一直抄
不過聽別人說碩士也差不多
建議是買一本筆記，省時也省力
不要跟著補習班進度走
自己看恐龍本慢慢K比較實際!

後面的習題加減看一下
考試越來越愛考這些


書籍:恐龍本、東缺西缺的上課筆記

觀念重要性:<span class="f1 hl">高</span> 題目重要性:<span class="f2 hl">中</span>

<span class="f3 hl">資結 &amp; 演算法:</span>

tree, graph, sort大重點一定要熟
然後很重要的是trace code的能力
君不見今年交大軟體考題多精美啊XD

資結一樣聖經本要弄一本
補習班講義常常只講一小角
真的有問題的時候還是只能查書

演算法讀得很不深
名校第五章後面全部放掉
但前面四章一定要看得很熟!!
考試常常直接叫你把程式碼默寫出來
進考場才開始想時間一定不夠

書籍:高銘講義三本、資結聖經本、演算法名校攻略

觀念重要性:<span class="f1 hl">高 </span>題目重要性:<span class="f4 hl">低</span>

<span class="b3 hl">1.幾乎沒有人能把書看完</span>

看了第一次，覺得題目我都會，考試應該沒問題
但看第二次的時候卻發現自己像得了失憶症一樣
甚麼都不記得，甚至連打過勾的題目都不記得怎麼寫
恭喜你，你很正常XD
大家都是這樣走過來的，撐下去就對了
一個考生的程度與他崩潰的次數成正比
情緒過了就收收心再出發!

<span class="b6 hl">2.盡全力增加自己的守備範圍</span>

有人說過考研究所就是基本題都寫對，難題寫對一題就上了
但問題是何謂基本題？對榜首來說每題可能都很基本
你該做的就是盡全力讓每一題對你而言都是基本
進考場前的日子就是在為自己擴增武裝
一進考場你有的就是那些其他就是臨場應變
千萬不要進考場發現自己只有一把小槍而別人都開坦克
那就等等乖乖被輾過去吧Orz

--

<span class="b2 hl">3.考古題無用論?</span>

準備到一個程度就該開始看考古題
但今年的考試如果全信考古題可能會被婊得很慘
台大的每一科,交大軟體..
不要有我寫超多年考古題我超強的觀念
考古題固然要寫，但還是講義上的東西比較重要
另外每科都要有一定的程度
不要想說把哪一科看到超強另一科爛掉沒關係
有個同學計組看到出神入化、OS放掉
但考台大的時候硬體題目發下來就發現麻煩大了

<span class="b5 hl">4.正面強化</span>

每天開始看書時創造一個儀式
讓淺意識習慣動作，會比較容易進入狀況
像是拍拍自己臉頰三下大喊一聲之類的
我自己的方法是每次看書前都看一次這個影片

<span class="b4 hl">5.大碩模擬考</span>

有些人可能會好奇我的排名
但其實第一次模考的時候我發現全部都不會寫
對每一題都有一點點記憶，但就是弄不出答案
當下就決定走人回家看書，考卷就直接丟老師桌上
如果有排名的話應該是無限大吧XD
模擬考的結果參考一下就好，事情沒有絕對

<span class="b2 hl">6.在考場內寫自己能力範圍內的題目</span>

考試不是每一題都要會，而是要至少比別人多寫對一題
你不會的題目，別人大概也不會(有看書的情況下)
這時的重點就是努力去鞏固你有把握寫對的題目
像交大的軟體我完全沒有看最後兩面的題目
只把前面簡單的題目鞏固好，這樣就夠了
台大的數學我也是花至少一半以上的時間做線代的暴力法
寫的多不如寫的精


--

<span class="b6 hl">7.用落榜的心情去考試</span>

考試最嘔的就是會的題目因為緊張沒想出來
所以最好不要有太多期待，但是也不要太消極
緊張不會讓你突然變強、但是卻會讓你變弱
另外不要相信自己的感覺，我寫清大跟成大時都覺得很順
結果清大落榜成大備91，反而是很抖的台大交大上了

<span class="b5 hl">8.有空就去考古題版看看</span>

不知道是不是巧合
我發現常常在考古題版解題的人，幾乎都是榜上有名
所以讀累了就去解解題，教學相長
幫助別人自己也會變強 何樂而不為?
像我就是比較爛只能解一些比較簡單的題目XD
如果逛遍所有文章發現沒幾個題目難得倒你
那恭喜你離上榜不遠了!
那邊也有很多前人的解題文是很棒的
我的OBST就是看ssccg大的文學會的

<span class="b1 hl">9.求的不是上榜 是一個問心無愧!</span>

不管原因是甚麼，你的確是打算加入這個戰場
在這段時間內目標就只有上榜，沒有其他
如果沒有這種覺悟，不如拿這段時間這做些更有生產力的事情
但也不是要讓自己繃得很緊，電池是需要充電的
如果有甚麼事情不做會後悔，去做，沒關係
但要給自己一個期限，時間過了就回來！
我11月花了兩個禮拜規畫某人的生日驚喜
也是弄完就馬上靜下心回到崗位上
當時的想法是就算最後沒上我也不打算拿這兩個禮拜當擋箭牌
不管做甚麼決定都要為自己負責任，這是最重要的一點。


打了一些沒甚麼營養的話

希望可以幫到以後的考生

也祝福所有人的努力都能開花結果!
