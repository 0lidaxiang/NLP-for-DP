<span class="b4 hl">三、準備考試</span>
<span class="b1 hl">(一)筆試</span>
<span class="f3 hl">1、看書</span>
<span class="f3 hl">(1)教科書部份</span>
<span class="hl">林鈺雄　刑法總則/李淑明　民法總則</span>
以上這兩本書很適合入門者看,解釋得很詳細而且用詞也不難。
準備考試期間我前前後後看了三次以上
(不過除了只有前兩次很認真琢磨，其他次則是想到才隨意拿起來翻翻= =++)。
我看書習慣念到一個段落自己再重新分析一次，這方法對我熟悉法律和筆試幫助很大。

<span class="hl">王澤鑑　民法總則</span>
王老師的書對我來說較硬(東西也太多)
因此我只有暑假期間認真念它,其餘時間當成睡前輕鬆讀物來翻XD
到考前我也沒有把它完整念完。
不過旁聽民總+債總課程時，王老師的書很適合做額外補充！

<span class="hl">法治斌董保成　憲法新論</span>
這本書到考前我念了2~3次。
雖然今年不考憲法，但書中概念很值得思考
<span class="f0 hl">(題外話：我覺得政大考題很喜歡道德倫理、平等自由等等的公法考題)</span>
另外,這本書也很適合初學者:)

<span class="hl">月旦法學教室</span>
休息時間有事沒事就去翻翻。
在看雜誌時可多多注意學者們寫文章的次序和前後文，
然後對特別有感覺的幾個要點筆記，學習他們寫申論的方法。

<span class="hl">補習班憲法筆記和公司法筆記</span>
這個其實可有可無，只因我同學恰好在準備律師司法官考試，所以有幸借用他的講義。
國考內容真的太難了，但其實真正需要的是大方向法感,念起來就不會有壓力。
考前我只大略聽一次錄音檔和翻閱兩次補習班筆記，寫政大社議分析有借用裡面的概念。


<span class="f3 hl">(2)課外讀物：</span>
<span class="hl">《大法官的智慧》</span>
書裡舉了很多爭議型案例很值得參考，例如墮胎、種族階級、隱私權等等問題。
雖然都是美國的案例，但可以藉此了解社會看法vs大法官們的觀點
是本值得一再鑽研的好書XD

<span class="hl">彼得．薩伯《洞穴奇案》</span>
這本書結合刑法、憲法、法哲學、法社會學、法倫理學等等。
這本書我前後翻了將近三次，不過每次看我頭都很昏覺得它在兜圈子……Orz

<span class="hl">David D. Friedman《經濟學與法律的對話》／熊秉元《法律經濟學開講》</span>
因為我是經濟背景的學生，所以決定用經濟分析法律。
若不是商管背景考生，倒是不用特別去看這兩本書~

《法律經濟學開講》針對經濟工具分析法律和經濟法律之間的異同有許多著墨
每看一次都會恍然大悟一次XD
《經濟學與法律的對話》則重視多層面的法律經濟分析，
我從書上學到很多經濟分析法律的訣竅。這本書用語簡單，看一看都會不小心笑出來XD


<span class="f3 hl">2、旁聽</span>
不得不推一下，旁聽是訓練自己增加法感最快的方法。
準備考試期間我旁聽了民總、刑總、債總和物權法。
雖說科法重視的是邏輯能力而非專業能力，但一些基本的名詞和概念也應該要知道，
例如：不當得利、罪行法定、阻卻違法事由等等族繁不及備載。
在應試時不需要寫出這些名詞，但用自己的語言說明的話會加很多分。
如果時間不夠，旁聽民總和刑總就綽綽有餘了。

但無論如何，其實聽過還是要用自己的話解釋一遍。
課後整理筆記時我會用經濟分析的方式再想一次，然後解釋給我法律系的朋友聽
吸收完才是自己的:)


<span class="f3 hl">3、思考爭議型問題</span>
法律和生活息息相關，平常有事沒事多看新聞或報紙然後對新聞做深入思考。
譬如我自己很喜歡刑事法律
所以看新聞時都會重新想一遍刑法構成要件該當、阻卻違法和有責性等等；
我也習慣邊設想小劇場再自我辯論XD
　　
另對於社會爭議大的議題也要多想幾次，而且每一次都要記錄要點以免重想浪費時間
<span class="f0 hl">(這真的很適合愛發呆的人XD)</span>
像是死刑、安樂死、墮胎等等的議題，我從刑總、民總、憲法、經濟層面分析，並且會把心得寫下來，每隔一段時間再回去看收穫都很多:)


<span class="f3 hl">4、時間規劃分配</span>
其實我念書時間很短又很混……Orz，
常常一天只念2~3小時甚至偶爾連續兩三天都不念書
<span class="f0 hl">(請不要學，因為容易產生罪惡感，讓自己心理狀況長期低迷)。</span>
不過我考前兩個月有做念書計畫表，也盡量跟上，所以考前大致都有念完。

我的忠告是：人不是鐵打的，要念書也要玩。
若一開始就規定自己要一天念10小時以上等等，不出一個月就會產生排斥情緒。
不如早睡早起，一週放一到兩天的假，其他時間全力念書
平衡身心之餘也讓念書變得有所期待:D


<span class="f3 hl">5、考古題</span>
我大概在考前一個月才開始寫考古題。
台大今年考題真的非常重視分析，以往的考古題其實很偏法律專業素養
<span class="f0 hl">(例如100年案例分析考科的兒少條例29條，根本是國考最愛考的題目之一…)</span>
所以看考古題時其實壓力不用太大，盡其所能書寫即可。
時間要分配好，基本上是依配分比重分配時間。

在寫題目時，盡量用自己的專業領域分析，而且可以用正反辯論，最後寫出結論即可。
今年台大考題，
第二題我再考卷上畫樹狀圖分類，再用財政學＋簡單的公平正義分析作答
第三題則是用到底誰要承擔風險和責任解釋。

寫題目和考試時都要注意排版和邏輯，
法律常用一、(一)、1、(1)的層次書寫
而且不要貪多，寫到第三層就綽綽有餘了<span class="f0 hl">(不然下場就會跟我一樣根本寫不完Orz)</span>

另外，我自己在考試時就當作是向教授分享我的想法XD，所以其實不太緊張。


<span class="f3 hl">6、小偏方──拜拜</span>
個人認為這是很有用的偏方XD。
信者恆信，至少去求個心安。
我覺得拜拜後冥冥之中得到祂們幫助才會寫得順利；
但若榜上有名記得還願，免得信用破產，所以許願時一定要量力而為XD。
拜拜重心誠，形式上倒是不用太苛求啦！



<span class="b1 hl">(二)口試</span>
一句話：了解自己真正想要的東西贏過證明自己能力很強。
<span class="f0 hl">而且你永遠不知道教授會喜歡什麼東西…Orz</span>
在準備口試一定要打草稿，我自己也會畫樹狀圖。
畫樹狀圖是因為你想到一個問題後還可以推敲想出下一個問題再下一個問題
同一類問題也可以擺在一起<span class="f0 hl">(就是建立問題模式和回應模式)</span>
但其實不用死背，熟練即可，以免別人看出你在念稿…冏。

千萬不要只有在腦裡沙盤推演就赴戰場，口試時只會腦中一片空白
多花一些時間手寫是值得的！

如同前面心得文所說的，要展現自信，面帶微笑，然後放輕鬆。
一樣當作把想法跟教授分享，太緊張反而會講不好。
我口試的時候就自說自話然後把場面弄得很歡樂，結束時覺得自己根本就是去搞笑的囧。

另外，面試者是有跡可尋的= =+++
通常會有所長、一位副教授、一位助理教授
所以面試前我會去看看當年度所內有哪些教授和其對應的領域
有機會就說說好話XD



<span class="b4 hl">四、結論</span>
之前一直聽前輩們說台大很保護自己的學生等等之類的話。
我覺得，其實不只是台大學生，所有的考生都是你的競爭者!
如果你不能證明你比其他考生更優秀或更適合念法律，那要怎麼要求教授們錄取你呢？
　　
另外，念法律教科書時不要把自己逼太緊，只要重視大方向就好
也不用看非常非常多的教科書，畢竟念不完只會更惶恐(重點是筆試還用不到……)。
反而應該要花更多時間思考自己想走的領域以及要如何利用所學和法律結合
若自己不能確立方向，又如何說服教授呢？

我相信會考科法所的人都是真心喜歡法律的！
相較用漂亮的自傳、學歷或其它能力證明自己，保持熱忱更加重要。
