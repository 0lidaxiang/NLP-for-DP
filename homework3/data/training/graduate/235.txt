<span class="f3 b1 hl">◎筆試準備：</span>

  我從考試前一年的四月開始念書，但因為那學期課有點多，
  沒有辦法專心念書，加上自己又常常時間管理不當，一直到
  六月中才痛改前非開始K書。


  <span class="f5 hl">※使用書籍：</span>

     經濟學：補習班上課講義、高勝銘經典題型、題庫班講義、歷屆試題、
            &lt;個體經濟學&gt;by楊雲明(工具書)、&lt;圖解通膨&gt;by王國強(睡前讀物)
     微積分：&lt;陳立微積分魔法書&gt;、歷屆試題、&lt;實用微積分&gt;by李德治(工具書)



以下把經濟學和微積分的讀書進度分開來說：
--
<span class="f2">※經濟學</span>
<span class="f3 hl">3~6月：基礎觀念期</span>

       就跟著補習班的進度，每個禮拜跟進，做整體全面性的吸收

<span class="f3 hl">7~9月：題目演練+觀念複習期</span>

       利用題目來加深和檢視學習內容的觀念以及細節，我是使用
       高勝銘的經典題型解析。
       我覺得這個時期是最痛苦的，每寫一題就覺得自己怎麼都不
       會，會有陷入題海中的感覺，覺得自己好笨明明都學完了怎
       麼寫題目就寫不出來。
<span class="f3 hl"> </span>花了三個月時間做完一遍的題目演練。

<span class="f3 hl">10~11月：筆記建立+第一次複習期</span>
<span class="hl"> </span>  演練題目(第二次)+講義，建立自己的整理筆記(盡量精簡但
      <span class="hl"> </span>  完整)，並且在第二次做題目的當中找出自己的弱點。十月底
       <span class="hl"> </span> 的時候我參加了題庫班，就跟著題庫班的進度進行另一輪複
        <span class="hl"> </span>習。這時期我有兩個複習進度，一個是自己的，另一個是補
         習班的。我自己的複習進度是在11月中完成第一次複習並完
         成筆記。
         之後就很少再把講義拿出來了，幾乎都是看自己的筆記在複習。

<span class="f3 hl">12月：第二次複習期</span>

      這時期我就看自己的筆記，一邊配合題庫班進度複習。當初
      訂的目標是當大家在快樂地跨年的那天我要完成第二次複習!
      結果有DELAY幾天，但整體而言有按時完複習XD

<span class="f0 hl">1月初：停擺期</span>
<span class="f0 hl">1月到1/18前讀書進度幾近停擺，因為期末考+預官考試的緣故。</span>
<span class="f3 hl">1月中~考前兩星期：考試模擬期</span>

                  安排一天一回考古題，計時80分鐘認真寫並計分。我的寫法
                  是一個禮拜專門寫一間，所以兩個半禮拜就寫三間，我當初寫
                  了台大商研、台大國企跟政大企管(較少回)

                  安排80分鐘的原因是預留考場緊張和塗畫卡的時間。

<span class="f3 hl">倒數兩週：最後衝刺期</span>

          經過歷屆試題演練，知道哪些是常考重點，趕快加強!!!!
          考前記得安排進度把重點考古題再看一遍!!
--

<span class="f2">※微積分</span>

       一開始我的念法是有點取巧，想靠寫考古題來抓重點，不過最後還是
       有乖乖把微積分的參考書寫完。而且全部都念過才是最快的準備方式
       。(除了某些幾乎沒考過的可以不念，例如向量積分的部分頂多考向量
       基本定理和Green's Theorem，但我當初還是都念了以防萬一。)

<span class="f3 hl">7~8月：基礎建立期</span>

       一開始真的甚麼都忘光光了。我先找了97~96的台大商研、台大
       國企考古題來寫。每題都不會XD。然後手邊放著一本參考書，
       努力拼湊出記憶裡的東西，往往一回考古題要做5個小時才做完。
       然後我會在”我覺得”是重點的單元上面打勾。這樣做的好處
       是有可能抓到讀書方向(但也有可能今年考題的方向無法捉摸orz)

       接下來的時間就開始把找到的重點單元念過一遍。(每天念1~5小時不等)


<span class="f3 hl">9~10月：乖乖啃書期</span>

       乖乖把整本參考書寫過一遍。我是用陳立的微積分魔法書(沒有
       補習)，大約一個月寫一本，上下剛好兩個月寫完。不過這時期
       我還是偷懶，跳過下冊某些困難的章節orz

<span class="f3 hl">11月：題海演練期</span>

      寫重點學校以外的考古題做練習，有點亂槍打鳥XD，大概一個
      禮拜做七~十回吧。
      有些學校考題方向比較不一樣，也算是一種弱點檢視法。發現
      自己有比較弱的章節時就回去讀參考書。

<span class="f3 hl">12月：自我挑戰期</span>

      傳聞寫台大微積分統一教學甲的考古題可以提升實力，於是乎我就挑戰了…
      結論是…這種方式會讓人深受打擊!!
      尤其是下學期的期末考，幾乎都是向量積分，統統不會!!!

      所以中肯的建議是，可以寫，但自己要能夠判斷哪些題目可以不用寫!!!
      有些難題不算超出範圍但很難，有時間挑戰的話還是可以做做看…

      另外，我也做了兩回台大的轉學考題目，覺得難度很適中，大家可以參考看看。

<span class="f3 hl">1月中~考試：考試模擬+最終複習期</span>

            基本上是配合經濟學，同樣一天一回，每回80分鐘。
            最近的幾年graduate版上都有口試門檻分數可以查，
            做個比較心中就有個底了。

            這時期我把微積分魔法書又做重點回顧
            (只做我覺得有必要做的題目)。

<span class="f3 hl">倒數兩週：最後衝刺期</span>

          配合筆記、重點式題目複習、演練考古題。對於考題
          方向心中應該已經有個底了，這時期就盡量找自己的
          弱點章節的題目來做!考前幾天就把公式確認複習!
          不需要再作特別難題!


總而言之，我覺得讀書進度每個人都應該<span class="b6 hl">依照自己的步調安排</span>，
適當地調整讀書計畫。讀書計畫難免有落後的時候，但只要持續
不斷努力到最後，一定會展現出成果的!

<span class="f3 b1 hl">◎口試分享</span>

  備審資料要認真做，尤其是制式化要求的一頁資料表要認真填寫，
  因為老師很有可能就只問其中的一項，然後一直猛問那一件事，所
  以<span class="b6 hl">每一項寫出來的東西都要很了解自己寫了甚麼!</span>

  另外，如果備審資料很認真地做了幾十頁，但教授只翻了第一頁，
  也不要太難過，因為我也有遇到這種情形，但又不能不認真做! Orz

  口試前，可以先找學長姐和同學來練習口試，在過程當中也可以從
  客觀條件上更加了解自己。一開始練習時光是自我介紹就會說不好，
  但就多講幾次，對自己的經歷很熟悉的話應該就沒有太大的問題。


以下分享各所的口試關卡，僅憑印象所以如果有遺漏還請見諒!

<span class="f2">※台大商研</span><span class="f0 hl">：每關六分鐘</span>
<span class="b2 hl">1. 個人關</span>：一開始先自我介紹，然後老師問一些相關問題，算是輕鬆的氣氛。

<span class="b2 hl">2. 英文關</span>：當場念稿一分鐘，念完翻譯。我也是抽到有關iphone的
           文章，整個看不懂=  ="
           翻譯完後老師會叫你把文章看完並說明這篇文章的大意。(不要看太久!)
           可是我根本來不及搞清楚這篇文章在說甚麼…每個字都
           看得懂但拼起來就不知道在幹嘛…於是就很緊張地亂七

<span class="b2 hl">3. 時事關</span>：我抽到的題目是：<span class="hl">「對於元宵節活動應不應該各縣市輪流舉辦的看法」</span>

           地方發展，並說明固定一個地點舉辦和輪流舉辦，對於

           接著老師問我該如何選擇場址？我說應該考慮交通方便、
           周遭建設、公平性…等等，很快就黔驢技窮了囧。
           最後有一點點擠盡腦汁地說應該考慮周邊地區的經濟發展
           ，然後老師笑著說時間到我可以出去了…

<span class="f2">※政大企管：</span>

因為政大有團體關，所以我們這一組9個人有先約出來認識，練習題目和默契!

<span class="f3 b1 hl">◎給未來準備考試的人的話：</span>

  準備研究所的期間，會有焦慮的情緒是很正常的，
  我有一陣子常常晚上失眠，白天卻又起不來，造成
  惡性循環，甚至十月的時候還天天睡到十點多……

  大概就是多找朋友聊聊天、互相加油打氣，然後隨時
  檢視自己的讀書進度並做調整。然後還有一個很重要
  的心態是：考試不是為了考試而考，進入研究所只是
  一個階段性的目標，往後的人生必定還會有更困難的
  挑戰!如果能夠這樣想，就能夠幫忙減輕一點壓力，
  因為知道這只是我們人生中的其中一個挑戰。

  而且<span class="b2 hl">我們能做的就是一直堅持到最後!! </span>
