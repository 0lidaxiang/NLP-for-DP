<span class="f4 b7 hl">求學歷程</span>
<span class="f2 hl">經濟(謝薇/王道):</span>

這科對於不是經濟系來說 剛開始很痛苦

因為範圍除了課本的範圍  還<span class="f5 hl">包括了國際形勢、管理、甚至基礎微積分</span>

所以在一剛開始我很挫折 上不到幾個章節 便想要放棄

但其實要掌握基本分並不難

當時是靠著我兩個強大(政大/交大)的同班同學救起來的

一有不會或是觀念不清 馬上問

今年我考的經濟我覺得沒有平常那麼難 未來<span class="f3 hl">考題趨勢應該就是基本功和時事</span>
<span class="f6 hl">選擇題和計算題</span>   基本理論熟悉 題目多看 應該一半拿的到

<span class="f6 hl">申論題和解釋名詞</span> 課本要背熟外  多練習培養手感

這樣來說經濟基本分一定拿的到!!!!!

至於進階的分數 要看個人的造化和領悟力了(小弟是笨的那群)



<span class="f2 hl">統計(陳浩/魏徵)</span>
<span class="f5 hl">統計這門對於考研究所來說是最好拿分的</span> 因為無所謂的主觀

所以除了解釋名詞 基本定理、攻式外 就是一直算!!!!!

當時學姊有給我建議我覺得很實用

<span class="f6 hl">把公式、導証全部整理在一本小筆記本裡面</span>

睡前、考前就是一直看筆記本

(可以把比較偏的東西寫再另外一本 學校上課無聊看)

剩下的就是多做考古題(現在的學校也都是抄來抄去)

<span class="f5 hl">很多學校漸漸越考越基本</span>   像今年台大 全部都是觀念

所以大家不要害怕統計 這會是好拿分的工具!!!!!

(交大、中央統計可能真的需要點實力 因為真的不簡單)



<span class="f2 hl">管理(李威/陳群)</span>

管理學 這門科目 大家往往覺得很主觀 但我覺得還是有方法

管理理論其實少於經濟太多太多  但和時事做結合是最難的

<span class="f1 hl">首先先挑定一本參考書 念到很熟很熟就對了</span>

然後另外建議大家每週抓個2~3小時去看看新聞、商周、哈佛、遠見(挑一本)

關心最新的時事  起初看不懂沒關係 有個印象就好

之後念完要自己去聯想哪個理論 哪種情形並且跟時事作結合

可以和同學互相分享時事、整理一張A4大小 無聊就可以翻一下

想必大家最害怕的無非是

管理學該怎麼寫才會獲得老師青睞?

李威老師(恩師)有所謂的寶典和十種解題方式

<span class="f6 hl">寶典是將自己題目的類型整理成完整的架構
</span><span class="f6 hl">
</span><span class="f6 hl">考前就只需要看這本 無法吸收的東西就不需要再放入</span>

解題方式說真的我並沒有全部都能靈活運用

但熟五、六種就可以輕鬆應對大部分的考題!!

舉例:APPLE對於全球的衝擊??

可以用邏輯式的思考方式(前因、後果)

驅動因素=&gt;APPLE=&gt;影響(衝擊)

(管裡沒有一定的答案 是看你有沒有正確的邏輯)

<span class="f5 hl">最近行銷、人資有越考越多的趨勢</span>

所以這些可能要自己額外準備或是請教老師


PS:有些人內容很多 但往往都有寫不完的情況

<span class="f1 hl">所以平常練習一定要設定時間、並且寫重點就好</span>

多餘的字眼能少則少 教授要改這麼多份 往往覺得你是屁話就是撇掉

<span class="f4 b7 hl">陳偉制度</span>
<span class="f2 hl">讀書計畫(每日和階段性)</span>

每個人手上都有一本小本子

到考前的每一天 每天唸什麼都要確實的寫上去

並且統計時數 以隨時警惕自己和檢視自己

過程:

我當時三下才開始補

3下          一周都是20小時以下

暑假(關鍵)   一周都是40~50小時

4上          一周都是50~60小時以上

寒假到考前   一周都是60~80小時


因為 很多人常常都從第一頁開始念

完全沒有規劃 常常熟的只有前面 然後就告訴自己:下次在念

所以需要時時檢視自己、並做調整


<span class="f2 hl">讀書會(考上的關鍵)</span>

組成讀書會 每週固定找一時間討論課業

每個人強的科目絕對不一樣 而且每個人的思考方式也都不一樣

不見得成績好的人答案就是對的

<span class="f1 hl">所以讀書會 可以提出自己想法 並且同組腦力激盪 互相鼓舞</span>

好的讀書會是一個未來成功的關鍵



<span class="f2 hl">輔導老師機制</span>

每個月都會有兩到三次的輔導時間

讀書會討論不出來的題目 就可以藉由此種方式來尋求解答

而且輔導老師多伴都是上一屆人稱所謂的強者

他們也當過學生 所以他們也了解問題的背後想法 並詳細解答



<span class="f2 hl">同進退的心</span>

陳偉有北中南三個點

師生感情都很好

會互相鼓勵、互相扶持  因為大家都想要一起上研究所

這個大家庭 大家有著共同的理想 所以準備起來不孤單



<span class="f3 hl">陳偉是私校最好翻身的地方  也是帶領走向名校的好地方</span>

因為管理很嚴格 讀書是用逼的

但 沒有付出 哪來的成功?

會偷懶卻想要考上好學校的人

陳偉是個不錯的選擇~
