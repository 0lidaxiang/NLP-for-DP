<span class="hl">零、前言</span>
過去這一年來，從專題、推甄、到考研究所，算是人生至目前最辛苦的一年。
無論生理或心理都有不小的壓力。在這期間不下數次想放棄很多事，但幸虧我的夥伴、
戰友、老師、家人，和這個版才能夠讓我一次次的調整好心態，不斷地再出發!

因此我在最後幾個月裡暗暗地想:假使我順利度過這一切，自己一定會好好幫助身邊和
我有類似煩惱或困難的人，不要讓他們跟我一樣感受到痛苦。
於是我先寫下了這篇心得文，希望這篇文章能夠帶你準備研究所的有用資訊。


<span class="hl">壹、背景與動機</span>
大一時準備過資管轉學考，有補習但可惜毅力不夠堅持到最後，就繼續待在淡江資管。
考研究所是因為在高中時對資訊領域有相對的興趣，大學又啟發了各項資管專業科目
的興趣，因此想繼續往碩班念，並且給自己一個往心目中前幾大學努力的目標。

從考前的一年多前就開始打探補習班、收集升學資訊和尋找志同道合的夥伴，自己補過
台北碩士完整的一年份資管全科班+題庫班。七月到九月密集做學校專題，十月和十一月
密集為了參加專題比賽和推甄做準備，當這一切結束時，知道自己中央沒推上(備不到的
地方)，整理好失落的情緒時已經準備聖誕節和跨年了...orz
之後到考完都是卯起來準備的。

<span class="hl">參、各科心得</span>
<span class="f4 hl">計概</span>
由於自己過去在大學學習計概都還蠻有熱誠的，因此這科還算有不少老本可以吃。
補習班的部分，老師會告訴同學要將計概拆開來念，也就是把每一科都獨立出來準備。
我的拆法是:
1. 基本計概(數字系統、計算機概論、數位邏輯、電腦新知...)
2. 作業系統(OS、系統程式、...)
3. 資料庫
4. 網路
5. 程式語言
(應該還有資結、但因為選考資結組就另闢大項目出來)

分為這五科，在獨力去收集書籍、考古題，搭配補習班的課程做一次完整徹底的複習，
過程中將大學沒學好跟學漏的都補起來，筆記對我非常重要，各科都有獨立出一份筆記。

其實因為從認真定下心來準備的時間不夠把理想的筆記做完，否則其實自己是認為筆記該
有基本的三種版本:

<span class="f6 hl">一、最粗略通包版</span>:包含所有重點與細節的筆記，無分熱門或冷門知識，內容有補習班
    老師的筆記和過去在學校學習的資料。筆記之外也包含大量補充書籍和相關的資訊
    來方便查找。

<span class="f6 hl">二、考前一個多月版</span>:其實是精簡化第一個版本而來的，包含自己常常搞錯及必定考出
    來的概念。差不多是在完整上完一次計概所拆出的那五科後，所整理出來的版本。
    範圍會縮小到自己報考學校的考古題範圍之上，做一次徹底的筆記整理。

<span class="f6 hl">三、考前沒幾天版</span>:這是為了考試期間針對各校，再次精簡第二個版本，補上該校特有的
    考型重點。這個版本已經精簡到十幾張紙就可以複習完一科，幾乎只有標題和自己
    想的內文和圖表。

筆記部分是貫穿整個考研生涯的關鍵成功因素之一，因為在複習或者寫題目時，都需時時
再回頭回憶起細節的知識，假如沒有寫在自己的筆記上，首先查找所花的時間會沒效率，
接著沒用自己的話寫下來會記不牢。

對於計概想要分享一個想法，考場上的一百分中比的是誰記得多又清楚，題目可能人人
都會個大概，但要正取就得靠平時不斷refresh所有知識，讓這些記憶體中的資料保持
著available的狀態。到了考前幾天和考試的期間，我幾乎都只在看最精簡版本的筆記，
看了標題後就閉起眼睛開始在腦中展開所有知識。筆記這項工作一直都幫了我很大的忙。


<span class="f4 hl">資結</span>
由於在學校只有修過一學期的資結課，自己的目標又在資結較看看重的學校，因此就選擇
補資工所的資料結構，在上洪逸的課時幾乎也都在抄筆記，要求一邊理解又要抄得快對
我來說有點吃力，因此上過完整的ㄧ輪課後，我也將資結的每個單元拆開來準備了。

資結的知識應該應該算是三科中最有範圍跟明確出題方向的一科，因此我認為這科的重點
在多算多活用，可以從不斷練習題目來培養，先求對在求快。

<span class="f4 hl">MIS</span>
MIS這科在大學的學習經驗非常的得過且過，因此再重新為了研究所而面對它時，其實
就跟第一次預見它一樣陌生。不過因為補習班的高進老師給的教材還有進度都很完整，
因此我的準備過程都是緊跟著它每周的進度而以，並沒有額外再多做什麼。

MIS也是很重筆記的科目，在當時高進老師也很強調筆記&amp;說出來的功夫要做足。因為MIS
就是一個很重視溝通面的科目，相較於計概資結而言，它活用得多而且也很容易被其它
資管背景的學生所輕視或者放棄。因此我認為重點在於閱讀書籍(培養sense)、筆記勤做
(將架構烙印在腦中)、多練習用嘴巴回答題目(多多活用，而且第二階段被問的機率非
常高)。



<span class="hl">肆、各校心得</span>
<span class="f6 hl">台科資管甲</span>
很特別的是這所學校在一般生的名額沒有筆試，因為自己過去在中央的推甄失利，對於
目標學校多數又有第二階段，因此就發奮的在十二月底前把書審都全部重寫了一次。
好處是之後其它學校的第一階段放榜後，到第二階段通常都只有三天到一個禮拜的時間，
準備書審及面試。假使一直到第一街學校放榜都還沒有書審，那麼接踵而來的放榜會帶來
另一波的忙碌高潮。所以建議可以報這一所，提早準備好書審資料，也更加認識自己
為什麼要考研究所。

<span class="f6 hl">清大資應乙</span>
今年是這個系所第一次將考科改成所有資管所考生都可以考的計概和英文，報名人數和
清大服科的加起來就差不多是今年全國資管好手的人數。
很特別的是今年不同以往兩年的考古題風格，基本題分數變得多，進階題占1/4，不過
也不用太擔心進階題，基本題的分數全拿到就有很高的希望正取。而基本題也全在補習
班的教材中。
而英文也不能忽視，資應將英文權重設為33.3%如果有心要用英文來補強計概，也是很好
的上榜策略。英文弱的想考這所必須提早準背英文，這是和其他資管所比較不一樣之處。

<span class="f6 hl">交大資管乙</span>
可惜這所備在中間，原本有將這所設定成前幾志願，但MIS寫偏掉就爆了。至於計概會考
很多很活的程式碼寫作，盡量將自己的想法寫出來可以多得到一點分，上榜關鍵也許就
在這些難題上了。
特別注意的是考英文只有五十分鐘，曾有聽過學長和同學以為跟其他科目一樣有一百分鐘
就慢慢地寫，最後來不及塗答案卡，這點要特別注意一下。
另外這所的第二階段的英文，和書審是一個很重要的項目，在考完交大筆試後也不能鬆懈!


<span class="f6 hl">中央資管丙</span>
中央是今年頭幾個考的，在考中央時早上五點半起來搭台鐵，精神和情緒壓力都蠻大的，
尤其是那段期間特別冷，回想起那段時間還蠻折磨的。

中央只有考計概和資結，題型都蠻結構化的，針對這兩科建議就是將考古題做到熟透。

計概部分:結構化的出題模式延續了五六年，今年這份是我覺得所有學校中最難的一份
計概考卷，不過一樣是要把握基本題，再多對一兩題進階題，正取的希望就非常大了。
另外要特別注意的是中央固定考25%的JAVA程式，必須額外針對這塊做準備，正取和備取
的差別或許會發生在此。

資結部分:同樣是非常結構化的考題，而且做考古題的過程會發現他有很多從往年考古題
出題的紀錄，因此<span class="f1 hl">一定</span>要做考古題，像我是往前做到94年，務必力求每
一題都要拿到八成以上的分數才行。


<span class="f6 hl">政大資管商管組</span>
政大在這幾年通常都是比較晚考的學校，因此需要額外的堅持力才能到最後。如果有心
想要針對政大來準備拼正取，其實堅持到最後就可以有蠻大的機會。
我覺得這所的關鍵在於筆試的高低，第二階段很難再把筆試的分數追回來或者爆掉。

計概部分:近兩年也開始考JAVA而不考C/C++，建議可以連同中央的考型一起準備，無論是
非題或者寫CODE都有幫助。資結的部分也越來越重，甚至今年還考了較冷門的skip list
和map(?)。多做考古題可以熟悉往後的出題模式。

MIS部分:也是和往年一樣很阿莎力的只出四題，這時候就得發揮高進老師的一套
"搬出架構"，用構面下去寫才能夠寫到適當的份量，也能讓批改的教授了解到你真的
懂MIS，我沒有特別為政大的MIS多做什麼準備，只有在考前暖好自己的手而已，因為從
考卷發下來到打鐘都是殺紅眼狀態的狂寫，就怕寫輸人。



<span class="hl">伍、補習班歷程與心得</span> (尋找-&gt;春季班-&gt;暑期班-&gt;秋季班-&gt;題庫班)

<span class="f6 hl">&lt;尋找&gt;</span>
差不多在三上快結束的時候就定下了自己會走考研究所的這條路，因此開始和同學打探情
報，情報來源不外乎是批踢踢、學長姐的口碑、同學間的情報、或是補習班外派來的人員
，由於自己是北部人，到台北車站很方便，當時就親自跑了高點、大碩、碩士，這三家比較
具有知名度的補習班，準備砸重金。多比較是好的，榜單、學費、師資、環境，這些是當
時所考慮的幾個重要項目。


<span class="f6 hl">&lt;春季班&gt;</span>
這段時間對我來說，是適應補習班與學校課業並行的日子，看看各科在考研究所上的全貌
，適應補習班老師的教法節奏，以及調適自己慢慢進入成為有考研心態的學生。
建議大家可以在此階段收集好考研究所的情報，像是選校、選所別、選考科等等，就如同
SA的規劃階段，畢竟越早確立自己的目標，冤枉路就越少。


<span class="f6 hl">&lt;暑期班&gt;</span>
暑期班是很多人強調的一段時間，大部分的正課都開在這時候，對於應屆生也正好有如此
完整的時間來增強實力。我在暑期各科開始的第一個星期，就和同學老師討論後決定轉考
資結組，從那時就完全確立自己是準備計概、資結、MIS。

在暑期班的過程中，由於學校專題的緣故，自己在上完課後已經沒有多餘的時間在去整理
或復習，因此這兩個多月我都是力求自己能夠在上課聽懂，做好筆記，下課問懂在離開。
建議是一定要再多做到預習跟複習，補習班所給的效益才能更加顯著。


<span class="f6 hl">&lt;秋季班&gt;</span>
秋季班的課程由於自己學校專題的發表及接續著代表學校比賽，外加準備推甄及四上課業
等等的緣故。資結和計概(OS)的課就沒有再去上，不過這兩科對於我已經在春季和暑期班
上過，內容也都是大同小異，因此只挑新的開課上。

建議這段時間可以將暑期班自己所遺漏或不足的地方補起來，或者挑弱點科目聽第二遍來
強化。當然這時候不像暑假有大量的完整時間，常常需要因為學校課業或者專題來分心，
這時候善用索碎時間來看自己的筆記其實是很重要的關鍵成功因素之一。


<span class="f6 hl">&lt;題庫班&gt;</span>
到了題庫班，所有的課都緊鑼密鼓著，秋季班的課要收尾，題庫班的課要開和加課，此時
還會遇上學校的期末考或者推甄面試的事情。不過題庫班大部分時間都是在四上的寒假開
始時才會進入密集期。

MIS是我在題庫班中得到最大收穫的科目，因為自己在暑期和秋季的MIS學習的都是分章節
的知識，對於整合性的思考和解題並沒有概念和經驗。因此這一塊我認為補習班老師帶給
我很高的效率。

網路與資料庫，其實這兩科就是帶來計概科目多且雜的強大助力之一，往往一間學校可能
偏重考網路或者資料庫(中央、交大)，一間學校卻完全不曾考過(政大)。因此藉由題庫班
的整理和猜題，能夠快速遞增進效率，將時間focus在刀口上。





<span class="hl">陸、綜合心得與結語</span>
準備研究所的過程其實是在比誰能夠堅持到最後，過程中有很多魔鬼想要將你拉到放棄
的陣營中，我曾經很脆弱的想放棄一切，但仔細回想這一切的努力，我並不想讓它白費。
不斷的給自己加油打氣，調整心態再去面對一切，如果你是正在努力考研的學弟妹，
加油!我想告訴你，這一切是值得的，做一件不會讓自己後悔的事情。
如果你感覺快要撐不下去了，不嫌棄的話歡迎寄信來聊聊，經過了這一切你會更強大。
