室友不夠權限 代PO

--------------------------------------------------------------------------------

板上比較少有關這組的相關資訊 PO一篇希望給以後的人參考

我是台大電子所丙組正取 在正取與備取邊緣

--------------------------------------------------------------------------------

<span class="f3">背景及動機</span>：

    私立長庚大學 電機系 應屆畢業生 排名：4/49

    對這組是從上學期初的EDA說明會有深入的認識

    聽完說明會後也對這組很感興趣 所以推甄也有推

    當時原本是想也同時準備這組的考試 有備無患

    但是不見棺材不掉淚

    等到推甄放榜看到自己備取很後面才開始準備


<span class="f3">應考科目</span>：

     邏輯設計、資料結構、還有一個很不重要的英文


<span class="f3">筆試準備</span>：

    一開始我先算了一下有多少時間可以準備大概是15周的時間

    時間不多 所以我大概把我的讀書分成三階段

    <span class="f6">速讀</span>：大概五周的時間 每天幾乎都是早上九點去圖書館晚上十二點半回寢室

          這一階段主要是把資料結構與邏輯設計讀完一遍

          並不要求自己能課本看過一次全部記住

          但是一定要知道碰到不會的問題要查課本哪章

          邏輯設計課本：張凡的計組與計結 電機專班的數位電路那本

          資料結構課本：Fundamentals of data structures in c

    <span class="f6">細讀</span>：大約也是五周的時間

          似乎是沒有體力 所以都是快中午吃完飯後才去圖書館一樣到晚上十二點半

          這一階段課本詳細看一遍  寫題目和習題看自己是不是真的會

          並且利用寫題目加深印象  這樣比較不容易忘

          其實這些課本也有中譯本  平時我是原文看不懂再看中譯本參考

          張凡的邏設是依照下面的第二本邏設課本出的  但是台大書單是第一本

          所以我兩本都有看  習題也有都寫過  後面的東西不大會考的我就沒有看

          邏輯設計課本：1.Fundamentals of Logic Design, 5th ed, Roth

                        2.Digital Design, 4/e, Morris

          資料結構課本：1.Fundamentals of data structures in c

                        2.資料結構分類題庫, 第五版跟第六版


    <span class="f6">複習</span>：剩下的五周 在這之間還遇到過年沒地方讀  在家裡幾乎沒碰到書

          前面三周資料結構我跟同學借題庫班的題目看 並且把筆記整理出來

          之後是資料結構都是看自己整理的筆記

          邏輯設計又把課本不熟的地方看過一遍

          最後兩周才開始寫考古題  台交政的資料結構考古題

          每一題都弄懂 也把板上提供的詳解再寫一遍


<span class="f3">口試準備</span>：

<span class="f1"> </span>因為推甄時就有準備口試  所以這面試這關比較好準備

    這邊特別提一下我口試的穿著 牛仔褲+白色襯衫

    當天每個人都試穿西裝整套的 但是我還是有正取

    所以其實穿著整齊  口試前總是會有人問 給大家參考

    教授們問題幾乎都是跟專題有關 還有修課上的專業科目相關問題
